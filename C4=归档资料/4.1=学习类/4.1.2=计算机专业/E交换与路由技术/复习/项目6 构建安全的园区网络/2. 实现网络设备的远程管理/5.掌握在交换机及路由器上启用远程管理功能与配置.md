在华为 VRP 体系中，路由器和交换机开启远程管理的命令**几乎是完全一致的**，唯一的区别仅仅在于“如何配置管理 IP 地址”。

下面为你整理了实操大题中能直接拿满分的“标准四步曲”：

### 第一步：打通网络底座（配置管理 IP）

设备必须有一个合法的 IP 地址，网管电脑才能 Ping 通它并建立连接。

**如果是路由器 (Router)：** 直接配置物理接口。

Plaintext

```
<Huawei> system-view
[Huawei] interface GigabitEthernet 0/0/0
[Huawei-GigabitEthernet0/0/0] ip address 192.168.1.254 24
[Huawei-GigabitEthernet0/0/0] quit
```

**如果是二层交换机 (Switch)：** 必须配置逻辑的 Vlanif 接口。

Plaintext

```
<Huawei> system-view
[Huawei] interface Vlanif 1
[Huawei-Vlanif1] ip address 192.168.1.254 24
[Huawei-Vlanif1] quit
```

---

### 第二步：建立 AAA 本地账号档案

无论是 Telnet 还是 STelnet，都必须在设备本地建一个有密码、有权限的网管账号。

Plaintext

```
[Huawei] aaa
! 创建名为 admin 的账号，设置密文密码，特权等级拉满至 15（大题采分点）
[Huawei-aaa] local-user admin password cipher Huawei@123
[Huawei-aaa] local-user admin privilege level 15
! 宣告该账号允许使用的接入服务：同时开启 Telnet 和 SSH 功能
[Huawei-aaa] local-user admin service-type telnet ssh
[Huawei-aaa] quit
```

---

### 第三步：配置 VTY 虚拟终端接口

VTY（Virtual Type Terminal）就是设备为你远程登录预留的逻辑“虚拟座位”。

Plaintext

```
! 进入 0 到 4 号虚拟终端（允许 5 个人同时登录）
[Huawei] user-interface vty 0 4
! 指定认证模式为刚才配置的 AAA 账号密码模式
[Huawei-ui-vty0-4] authentication-mode aaa

! 【考点分水岭：协议放行】
! 如果考题只要求 Telnet：
[Huawei-ui-vty0-4] protocol inbound telnet

! 如果考题要求更安全的 STelnet (SSH)：
[Huawei-ui-vty0-4] protocol inbound ssh
[Huawei-ui-vty0-4] quit
```

---

### 第四步：STelnet (SSH) 的专属加分项

如果考题明确要求使用**安全加密**的远程登录（STelnet），在完成前三步之后，必须补齐下面两行代码，否则绝对无法连通，这是考卷中最容易漏掉的暗雷。

Plaintext

```
! 1. 必敲命令：生成本地 RSA 密钥对（用来给数据高强度加密）
[Huawei] rsa local-key-pair create
(此时系统会提示你选择密钥长度，直接按回车键默认确认即可)

! 2. 必敲命令：全局开启 SSH 服务（部分版本默认关闭）
[Huawei] stelnet server enable
```

---

### 考场终极验证指令

如果考卷要求写出验证连通性的方法，或者你在模拟器上需要排错，这两条指令非常管用：

- **测试连通性：** 在测试终端（PC）的命令行输入 `telnet 192.168.1.254` ；如果是 SSH，则输入 `ssh client first-time enable`（首次登录认证）后，再输入 `stelnet 192.168.1.254`。
    
- **查看当前谁在线：** 在路由器/交换机上敲击 `display users`，就能清晰看到当前是谁通过哪个 VTY 接口登录在了这台设备上。
