# 项目2 运用交换机构建小型园区网络

## 1. 交换机基本配置

### 识记：基本概念
*   **交换机管理方式**：
    *   **本地管理（带外管理）**：通过 Console 线连接电脑的串口和交换机的 Console 端口。不需要网络连接，常用于初始配置。
    *   **远程管理（带内管理）**：通过 Telnet 或 SSH 协议，跨越网络对交换机进行配置。前提是交换机已配置 IP 地址且网络可达。
*   **Console 端口登录管理**：
    *   物理连接：Console 线（反转线）。
    *   参数设置：波特率 **9600 bps**，无流控，8 位数据位，1 位停止位。
*   **交换机命令视图模式**（以华为 VRP 系统为例）：
    *   **用户视图** `<Huawei>`：只能查看基本运行状态，不能修改配置。
    *   **系统视图** `[Huawei]`：使用 `system-view` 进入，可进行全局性的系统配置。
    *   **接口视图** `[Huawei-GigabitEthernet0/0/1]`：进入具体端口进行参数修改。

### 领会：技术细节
*   **交换机端口类型**（VLAN 核心）：
    *   **Access（接入端口）**：连接电脑（终端）。只属于 1 个 VLAN，收到不带标签的数据包后打上 PVID 标签。
    *   **Trunk（干道端口）**：连接交换机或路由器。允许传输多个 VLAN 的数据，数据包带着 VLAN Tag 传输。
    *   **Hybrid（混合端口）**：华为默认模式。既可以接电脑也可以接交换机，灵活性最高，可手动控制哪些 VLAN 打标，哪些不打。
*   **交换机端口双工模式**：
    *   **全双工 (Full-Duplex)**：可以同时发送和接收数据（类似电话）。
    *   **半双工 (Half-Duplex)**：同一时刻只能发或只能收（类似对讲机）。
*   **交换机端口速率**：
    *   指每秒传输的比特数。常见有 10M、100M（百兆）、1000M（千兆）。

### 应用：基础配置
在 eNSP 模拟器中，你可以通过以下命令完成基础配置：

```bash
<Huawei> system-view                 # 进入系统视图
[Huawei] sysname SwitchA             # 修改交换机名称为 SwitchA
[SwitchA] vlan 10                    # 创建 VLAN 10
[SwitchA] interface GE0/0/1          # 进入千兆口 0/0/1
[SwitchA-GigabitEthernet0/0/1] description link_to_PC1  # 添加端口描述
[SwitchA-GigabitEthernet0/0/1] port link-type access    # 设置为 Access 类型
[SwitchA-GigabitEthernet0/0/1] port default vlan 10     # 加入 VLAN 10
[SwitchA-GigabitEthernet0/0/1] quit                     # 退出当前视图
[SwitchA] display vlan               # 查看 VLAN 信息
[SwitchA] save                       # 保存配置
```

**理解小贴士：**
你可以把交换机的“视图”想象成“房间”：
- `用户视图` 是客厅（只能看）。
- `系统视图` 是书房（可以改全局设置）。
- `接口视图` 是书房里的特定抽屉（只能改那个抽屉的东西）。
