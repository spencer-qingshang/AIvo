# 习近平新时代中国特色社会主义思想概论 - 第九章

## 📚 考核知识点与核心笔记

### 1. 坚持中国特色社会主义法治道路

> **🎯 官方大纲要求**
> *   **识记**：
>     *   **唯一正确道路**：中国特色社会主义法治道路。
>     *   **核心要义**：坚持党的领导、中特制度、贯彻中特法治理论。
>     *   **最根本保证**：中国共产党的领导。
>     *   **根本目的**：依法保障人民权利。
>     *   **习近平法治思想**：全面依法治国的根本遵循。
> *   **领会**：全面依法治国是坚持和发展中特社会主义的**本质要求和重要保障**。
> *   **应用**：
>     *   **深刻革命**：全面依法治国是国家治理的一场深刻革命。
>     *   **五大原则**：党领导、以人民为中心、法律面前人人平等、依法治国与以德治国结合、从中国实际出发。

#### (1) 为什么是“深刻革命”？（应用）
*   **地位**：法律是治国之重器，法治是国家治理体系和治理能力的重要依托。
*   **作用**：关系党执政兴国、人民幸福安康、党和国家长治久安。
*   **保证**：为“制度之治”提供最基本、最稳定、最可靠的保障。

#### (2) 核心要义与原则（识记/应用）
*   **核心要义**：三位一体（党的领导 + 中特制度 + 法治理论）。
*   **五个原则（必背）**：
    1.  坚持**中国共产党的领导**（法治之魂）。
    2.  坚持**以人民为中心**（法治的根本立场）。
    3.  坚持**法律面前人人平等**（基本属性）。
    4.  坚持**依法治国和以德治国相结合**（法安天下，德润人心）。
    5.  坚持**从中国实际出发**（突出中国特色、实践特色、时代特色）。

### 2. 建设中国特色社会主义法治体系

> **🎯 官方大纲要求**
> *   **识记**：
>     *   **总目标**：建设中国特色社会主义法治体系、建设社会主义法治国家。
>     *   **总抓手**：建设中国特色社会主义法治体系。
>     *   **依宪治国/执政**：坚持依法治国首先要坚持依宪治国。
>     *   **党内法规体系**：中国特色法治的鲜明特征。
> *   **领会**：坚持依宪治国、依宪执政。
> *   **应用**：更好推进法治体系建设（五个子体系）。

#### (1) 为什么说它是“总抓手”？（识记）
*   依法治国各项工作都要围绕这个“总抓手”来谋划和推进。
*   它是国家治理体系的**骨干工程**。

#### (2) 五大子体系（应用·重点）
1.  **完备的法律规范体系**：以宪法为核心，立善法于天下。
2.  **高效的法治实施体系**：执法、司法、守正各环节协调运行。
3.  **严密的法治监督体系**：以规范和约束公权力为重点。
4.  **有力的法治保障体系**：政治、思想、组织、人才、科技保障。
5.  **完善的党内法规体系**：以党章为根本，体现“依规治党”。

### 3. 加快建设法治中国

> **🎯 官方大纲要求**
> *   **识记**：
>     *   **总体目标**：法律规范科学完备、执法司法公正、权利保障充分、法治信仰确立。
>     *   **完善法律体系**：以宪法为核心。
> *   **领会**：**工作布局**（四个一体推进、一个一体建设）。
> *   **应用**：建设更高水平的法治中国。

#### (1) 工作布局（领会·必背）
*   **四个共同推进**：依法治国、依法执政、依法行政。
*   **一个一体建设**：法治国家、法治政府、法治社会。
*   **统筹内外**：统筹推进国内法治和涉外法治。

#### (2) 四大重点任务（应用）
1.  **科学立法**：提高立法质量，以良法促进发展、保障善治。
2.  **严格执法**：法治政府建设是“主体工程”，做到严规范、公文明。
3.  **公正司法**：维护社会公平正义的“最后一道防线”。
4.  **全民守法**：使法治成为全民的信仰，建设法治社会。
