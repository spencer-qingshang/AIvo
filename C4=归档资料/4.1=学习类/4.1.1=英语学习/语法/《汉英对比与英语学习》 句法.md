# 主语

英语句子里主语时不能少了的，可以省略，比如祈使句。而且很容易找到，一般都在句首，极少数倒装和there be句型放在后面。

汉语可以没有主语，但是转化成英语则需要自己增补进主语。

# 英语形式主语it

如果汉语的主语相当长，不是一个词，而是词组或句子，同时谓语比较短，那么转化时就可以考虑用it做先行主语。

# 汉语人称主语和英语物称主语

英汉之所以主语不同，还是动词起的作用，动词和前面的主语以及后面的宾语形成固定搭配，正是这些特殊的语义关系使物称主语成为可能。

# 汉语偏句与英语主语

汉语的偏句相当于英语的从句。汉语转化时，如果有“因为”“由于”类似的连狙，或者学生能明显察觉到偏句和正句之间的因果关系，就会立刻想到because，于是不假思索地将汉语的结构和主语搬进英语。其实，英语表达编一个方式来说或更好，就是用物称主语。

# 汉语多个主语和英语单一主语

汉译英时要注意不要被汉语的多个主语干扰，英语只有单一主语。

汉语也可以随意省略主语，但在译为英语时需要清清楚楚表达出来。

# 汉英主语转换

多练习下，一句中文可以转化成多少句英文，这个训练方法可以快速发现英汉结构差距。

# 谓语

英语以动词为中心，谓语都可以称作动词性谓语。汉语的谓语很复杂。

当汉语的动词特别多的时候，要分析表达的内容。

第一是表示两个动作时间的先后，这个用and连起来

# 宾语

汉语没有及物动词，使用起来很复杂

# 定语

汉语的定语很灵活，只要加上......的就行。

英语中很长的定语从句很难处理，注意换换语序，汉语不需要那么长的定语，多加点逗号吧。

# 同位语

汉语里同位语用的很少，表现也单位，汉语的同位语一般常在中心词前面，英语的则有许多放在中心词后面。

# 状语

英语里的状语，转化成汉语不是状语，而是补语。
