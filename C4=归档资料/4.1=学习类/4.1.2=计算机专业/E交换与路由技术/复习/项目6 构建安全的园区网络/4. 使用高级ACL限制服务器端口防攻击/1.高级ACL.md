如果说基本 ACL 是只查“原籍”的普通门卫，那么**高级 ACL（编号 3000-3999）**就是配备了 X 光机、拥有最高查验权限的“机场特级安检”。

在《交换与路由技术》的考试中，高级 ACL 绝对是综合配置大题里最核心的压轴武器。它不仅能精准打击特定主机的特定行为，还能最大程度地保护网络带宽。

要想在考卷上把高级 ACL 拿捏得死死的，你必须掌握它的“五元组”雷达、部署原则以及实战配置代码流：

### 一、核心查验机制：五元组（5-Tuple）雷达

基本 ACL 只能查源 IP，而高级 ACL 能够同时透视数据包的五个核心参数。只要这五个参数有一项不符合你的规定，直接拦截！

1. **源 IP 地址 (Source IP)：** 谁发出的？
    
2. **目的 IP 地址 (Destination IP)：** 要发给谁？
    
3. **协议类型 (Protocol)：** 用的是 TCP、UDP 还是 ICMP（Ping）？
    
4. **源端口号 (Source Port)：** 发送端用哪个端口？（通常随机，很少查）
    
5. **目的端口号 (Destination Port)：** 接收端提供什么服务？（如 80 是网页，23 是 Telnet，21 是 FTP，**考试极高频考点**）。
    

### 二、部署灵魂原则：尽可能“靠近源端 (Source)”🌟

这是选择题和排错题最爱挖的坑！

- **为什么靠近源端？** 高级 ACL 极其精准，不会出现“误杀”情况。既然它能精准识别出非法的恶意流量（比如某员工上班时间看视频的流量），那就应该**在这些垃圾报文刚刚离开电脑、进入网络的第一时间（也就是连接该员工的那台接入层交换机或路由器接口上）把它干掉**。
    
- **好处：** 避免了垃圾流量在全网穿梭，白白占用骨干链路的宝贵带宽。

### 三、实战演练：大题满分代码流

**考试最经典场景（允许业务，拒绝 Ping）：**

公司研发部 PC 1（`192.168.1.10`）和财务部服务器（`192.168.2.100`）在不同网段。

**要求：** 出于安全考虑，禁止 PC 1 Ping 通服务器，但必须允许 PC 1 访问服务器的 Web 网页服务（TCP 80 端口）。

**第一步：创建高级 ACL 并精准定点清除**

Plaintext

```
<Huawei> system-view
! 1. 创建编号为 3000 的高级 ACL
[Huawei] acl 3000

! 2. 封杀 Ping 报文（协议为 icmp，源和目的都用全 0 通配符精确锁定单台设备）
[Huawei-acl-adv-3000] rule 5 deny icmp source 192.168.1.10 0.0.0.0 destination 192.168.2.100 0.0.0.0

! 3. 放行 Web 网页报文（协议必须写 tcp，并用 eq 80 指定目的端口）
[Huawei-acl-adv-3000] rule 10 permit tcp source 192.168.1.10 0.0.0.0 destination 192.168.2.100 0.0.0.0 destination-port eq 80

[Huawei-acl-adv-3000] quit
```

_(💡 考点解析：在高级 ACL 中，如果指定了端口号 `eq 80`，前面的协议关键字绝不能写 `ip`，必须明确写出 `tcp` 或 `udp`，因为端口是传输层特有的概念。)_

**第二步：找准大门，方向应用（Inbound）**

根据“靠近源端”的原则，我们来到**连接 PC 1 的那台路由器/交换机**上。假设连接 PC 1 的网关接口是 `GigabitEthernet 0/0/1`。

此时数据包是从 PC 1 发出，刚刚**进入**网关接口，所以方向是 `inbound`。

Plaintext

```
[Huawei] interface GigabitEthernet 0/0/1
! 在入方向调用高级 ACL 3000
[Huawei-GigabitEthernet0/0/1] traffic-filter inbound acl 3000
[Huawei-GigabitEthernet0/0/1] quit
```

### 四、考场通配符与端口号的“隐形地雷”

在卷面上写高级 ACL 时，这三个细节最容易被扣分：

1. **Any 关键字的滥用：** 如果你想表示“拒绝 192.168.1.10 访问所有人”，目的 IP 必须写成 `destination any`，而不能省略不写。
    
2. **EQ 关键字：** 匹配特定端口时，必须加 `eq`（等于），例如 `destination-port eq 80`。除了 `eq`，考卷偶尔也会出现 `gt`（大于）或 `lt`（小于），但 `eq` 占据了 95% 的考点。
    
3. **常见端口号必须背熟：**
    
    - **TCP 80**：HTTP (Web 网页)
        
    - **TCP 20/21**：FTP (文件传输)
        
    - **TCP 23**：Telnet (远程登录)
        
    - **TCP 22**：SSH (安全远程登录)
        
    - **UDP 53**：DNS (域名解析)
