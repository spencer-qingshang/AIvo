### 一、第一阶段：网络安全“图纸”规划（大题文字论述得分点）

千万别上来就敲击键盘，考卷上通常要求你先写出规划思路：

1. **区域与物理边界规划（划地盘）：**
    
    - 明确核心设备：内网用二层/三层交换机，边界用防火墙/出口路由器。
        
    - 划定安全区域：员工办公网划入 **Trust（信任区）**，对外的门户网站服务器划入 **DMZ（隔离区）**，连接外网的接口划入 **Untrust（非信任区）**。
        
2. **身份与设备管理规划（保主帅）：**
    
    - 绝不允许任何密码明文传输。广域网专线必须规划 **PPP CHAP** 认证。
        
    - 设备远程管理必须规划 **AAA 认证体系**，禁用 Telnet，改用高安全的 STelnet (SSH)。
        
3. **流量管控与地址规划（设关卡与穿马甲）：**
    
    - **NAT 规划：** 内网上网规划 **Easy IP（NAPT）**，节约公网地址并隐藏内网拓扑；外网访问官网规划 **NAT Server**，只暴露出 80/443 端口。
        
    - **安全策略/ACL 矩阵规划：** 默认全网拒绝（Deny All）。只写 Permit 规则：允许 Trust 访问 Untrust；允许 Untrust 访问 DMZ 的 Web 服务；绝对禁止 Untrust 主动访问 Trust。

### 二、第二阶段：终极实战满分代码流（综合配置）

假设考卷最后一题给出一个综合拓扑：你需要在一台华为出口设备（兼具路由与防火墙功能）上，把上面规划的安全防线全部落地。

#### 第一道防线：设备自身的金钟罩（AAA 与登录安全）

Plaintext

```
<Huawei> system-view
! 1. 开启本地 AAA 数据库，建立最高权限网管账号（必须密文）
[Huawei] aaa
[Huawei-aaa] local-user root password cipher Admin@2026!
[Huawei-aaa] local-user root privilege level 15
[Huawei-aaa] local-user root service-type ssh terminal
[Huawei-aaa] quit

! 2. 锁死 VTY 远程接口，只允许最高安全的 SSH 拨入
[Huawei] user-interface vty 0 4
[Huawei-ui-vty0-4] authentication-mode aaa
[Huawei-ui-vty0-4] protocol inbound ssh
[Huawei-ui-vty0-4] quit
```

#### 第二道防线：广域网防窃听（专线接口 CHAP 认证）

Plaintext

```
! 假设 Serial 4/0/0 是连接另一个城市分部的专线（作为主认证方）
[Huawei] interface Serial 4/0/0
[Huawei-Serial4/0/0] link-protocol ppp
[Huawei-Serial4/0/0] ip address 10.1.1.1 24
[Huawei-Serial4/0/0] ppp authentication-mode chap
[Huawei-Serial4/0/0] quit
```

#### 第三道防线：内网隐身衣（NAT Easy IP 部署）

Plaintext

```
! 1. 用基本 ACL 抓取允许上外网的员工网段 (192.168.1.0/24)
[Huawei] acl 2000
[Huawei-acl-basic-2000] rule 5 permit source 192.168.1.0 0.0.0.255
[Huawei-acl-basic-2000] quit

! 2. 在连接互联网的公网接口 (G0/0/1) 上，开启 NAT 转换
[Huawei] interface GigabitEthernet 0/0/1
[Huawei-GigabitEthernet0/0/1] ip address 200.1.1.1 24
[Huawei-GigabitEthernet0/0/1] nat outbound 2000    ! 核心：调用 ACL 执行 Easy IP
[Huawei-GigabitEthernet0/0/1] quit
```

#### 第四道防线：基于安全区域的终极裁决（安全策略）

Plaintext

```
! 1. 将物理接口划入对应的安全区域
[Huawei] firewall zone trust
[Huawei-zone-trust] add interface GigabitEthernet 0/0/0   ! 内网接口加入信任区
[Huawei-zone-trust] quit
[Huawei] firewall zone untrust
[Huawei-zone-untrust] add interface GigabitEthernet 0/0/1 ! 外网接口加入非信任区
[Huawei-zone-untrust] quit

! 2. 编写放行策略：允许 Trust 区域访问 Untrust 区域
[Huawei] security-policy
[Huawei-policy-security] rule name Allow_Internet
[Huawei-policy-security-rule-Allow_Internet] source-zone trust
[Huawei-policy-security-rule-Allow_Internet] destination-zone untrust
[Huawei-policy-security-rule-Allow_Internet] action permit
[Huawei-policy-security-rule-Allow_Internet] quit
```
