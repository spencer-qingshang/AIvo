### 三、 简答题 (5 x 6分 = 30分)

31. **简述路由器的工作原理。**
    路由器收到数据包后，解封装链路层头部，提取目的 IP 地址；然后在路由表中进行匹配（最长匹配原则）；匹配成功后，重新封装新的链路层头部，从相应的出接口转发出去。

32. **什么是“单臂路由”？它解决了什么问题？**
    单臂路由是指在路由器的一个物理接口上配置多个逻辑子接口，每个子接口对应一个 VLAN，作为其网关。它解决了二层交换机无法实现跨 VLAN 通信的问题，节省了路由器的物理端口。

33. **简述 DHCP 的两种配置方法（接口地址池与全局地址池）的区别。**
    - 接口地址池：配置在特定接口下，仅供该接口所在的网段使用，配置简单。
    - 全局地址池：在系统视图下创建，可以供路由器上的多个接口、甚至通过中继供远程网段使用，灵活性更高。

34. **简述静态路由与动态路由的区别及其各自的适用场景。**
    - 静态路由：手动配置，不占用带宽，安全可控，适用于小型网络或网络边缘。
    - 动态路由：协议自动学习，适应拓扑变化快，配置复杂需消耗 CPU/带宽，适用于中大型网络。

35. **简述 OSPF 划分区域（Area）的作用是什么？为什么 Area 0 被称为骨干区域？**
    划分区域可以减小 LSDB（链路状态数据库）的大小，降低路由器内存和 CPU 的消耗，并将拓扑变化限制在区域内。Area 0 是骨干区域，所有非骨干区域的流量必须经过 Area 0 转发，以防止区域间产生环路。

### 四、 综合应用题 (2 x 5分 = 10分)

36. **静态路由与浮动路由配置 (5分)**：
    路由器 R1 连接着 R2，要求配置到达目的网段 172.16.1.0/24 的路由：
    (1) 主路径下一跳为 10.1.1.2。
    (2) 备份路径下一跳为 20.1.1.2，优先级设为 100。
    **请写出 R1 的配置命令。**

```shell
ip route-static 172.16.1.0 255.255.255.0 10.1.1.2
ip route-static 172.16.1.0 255.255.255.0 20.1.1.2 preference 100
```

37. **OSPF 基础配置 (5分)**：
    在路由器 R1 上启动 OSPF 进程 1，Router-ID 为 1.1.1.1，并将接口 GE0/0/0 所在的网段 192.168.10.0/24 宣告到区域 0 中。
    **请写出对应的配置命令。**

```shell
ospf 1 router-id 1.1.1.1
 area 0
  network 192.168.10.0 0.0.0.255
```
