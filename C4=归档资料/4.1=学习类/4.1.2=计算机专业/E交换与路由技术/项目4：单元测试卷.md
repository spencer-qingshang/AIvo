### 三、 简答题 (5 x 6分 = 30分)

31. **简述 IPv6 相比于 IPv4 的主要优势（列举四点）。**
    (1) 近乎无限的地址空间；(2) 简化的报文头部，提高转发效率；(3) 支持无状态地址自动配置（SLAAC）；(4) 内置 IPsec 安全支持；(5) 更好的 QoS 支持。

32. **简述 IPv6 地址的缩写规则。**
    (1) 每组中的前导“0”可以省略；(2) 连续的、全为“0”的组可以用双冒号“::”代替；(3) 在一个 IPv6 地址中，双冒号“::”只能出现一次。

33. **简述 RIPng 与 RIPv2 的主要区别。**
    (1) 网络环境：RIPng 运行在 IPv6 协议栈，RIPv2 运行在 IPv4；(2) 配置方式：RIPng 在接口下开启，RIPv2 通过 network 宣告网段；(3) 组播地址：RIPng 使用 FF02::9，RIPv2 使用 224.0.0.9；(4) 端口：RIPng 使用 UDP 521，RIPv2 使用 UDP 520。

34. **什么是“汇总路由”？在 IPv6 网络中配置汇总路由有什么实际意义？**
    汇总路由是将一组具有相同前缀的连续路由条目合并成一条。意义：减小路由器的路由表规模，降低 CPU 和内存消耗；屏蔽网络拓扑变化带来的抖动，提高网络稳定性。

35. **简述 OSPFv3 与 OSPFv2 的异同点。**
    相同：核心算法（SPF）、区域架构、报文类型（Hello, DD, LSR, LSU, LSAck）基本一致。
    不同：OSPFv3 基于链路运行而非网段；OSPFv3 支持 IPv6 地址族；OSPFv3 的 Router ID 仍为 32 位；配置方式移至接口视图下。

### 四、 综合应用题 (2 x 5分 = 10分)

36. **IPv6 静态路由配置 (5分)**：
    要求在路由器 R1 上配置一条静态路由，使得网络能够访问目的网段 `3001::/64`，下一跳 IPv6 地址为 `2001::2`。
    **请写出对应的华为配置命令。**

```shell
ipv6
ipv6 route-static 3001:: 64 2001::2
```

37. **OSPFv3 基础配置 (5分)**：
    在路由器 R1 上启动 OSPFv3 进程 1，手动配置 Router-ID 为 `1.1.1.1`，并将接口 `GigabitEthernet 0/0/0` 加入 OSPFv3 的区域 0。
    **请写出对应的配置命令。**

```shell
ospfv3 1
 router-id 1.1.1.1
interface GigabitEthernet 0/0/0
 ospfv3 1 area 0
```
