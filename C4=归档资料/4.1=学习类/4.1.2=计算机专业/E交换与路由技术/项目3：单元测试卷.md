### 一、 单项选择题 (20 x 2分 = 40分)

1. 在华为路由器上，从用户视图（User View）切换到系统视图（System View）应使用的命令是（ ）
   A. system-view
   B. system
   C. sysname
   D. interface

2. 路由器工作在 OSI 参考模型的哪一层？（ ）
   A. 物理层
   B. 数据链路层
   C. 网络层
   D. 传输层

3. 连接 PC 的串口与路由器的 Console 端口进行本地配置时，通常使用什么线缆？（ ）
   A. 直通线
   B. 交叉线
   C. 反转线 (Rollover)
   D. 光纤

4. 在华为设备中，设置设备名称为 "R1" 的正确命令是（ ）
   A. hostname R1
   B. name R1
   C. sysname R1
   D. set name R1

5. 路由器接口的默认状态通常是（ ），需要手动开启。
   A. Up
   B. Shutdown (Down)
   C. Active
   D. Forwarding

6. 下列哪种接口类型通常用于广域网（WAN）的高速同步串口连接？（ ）
   A. Ethernet
   B. Serial
   C. GE
   D. Console

7. DHCP 服务中，由服务器向客户端确认 IP 地址分配成功的报文是（ ）
   A. Discover
   B. Offer
   C. Request
   D. ACK

8. 若要让路由器接口 GE0/0/0 直接作为 DHCP 地址池网段，应在接口下配置命令（ ）
   A. dhcp select global
   B. dhcp select interface
   C. ip pool interface
   D. dhcp enable

9. 在全局地址池配置中，用于排除某些已占用地址不被分配的命令是（ ）
   A. excluded-ip-address
   B. static-bind
   /C. lease
   D. network

10. 单臂路由技术中，路由器物理接口下创建的逻辑接口被称为（ ）
    A. 虚接口
    B. 隧道接口
    C. 子接口
    D. 环回接口

11. 实现单臂路由时，子接口必须配置（ ）命令来关联特定的 VLAN 标签。
    A. dot1q termination vid
    /B. vlan batch
    C. port link-type
    D. encapsulation

12. 为使单臂路由子接口能够正常处理 ARP 请求，必须手动开启（ ）功能。
    A. arp broadcast enable
    B. ip address
    C. shutdown
    D. routing

13. 静态路由的默认优先级（Preference）在华为设备上是（ ）
    A. 0
    B. 60
    C. 100
    D. 120

14. 路由表中，当到达同一目的网段存在多条路径且优先级相同时，会发生（ ）
    A. 路由环路
    B. 负载分担 (ECMP)
    C. 主备备份
    D. 路由抑制

15. 命令 `ip route-static 0.0.0.0 0.0.0.0 10.1.1.1` 配置的是什么路由？（ ）
    A. 静态路由
    B. 默认路由
    C. 动态路由
    D. 直连路由

16. 浮动路由（Floating Route）是通过修改静态路由的（ ）来实现路径冗余的。
    A. 目的地址
    B. 下一跳
    C. 优先级 (Preference)
    D. 出接口

17. RIPv2 协议发送更新报文的目的地址是组播地址（ ）
    A. 224.0.0.1
    B. 224.0.0.5
    C. 224.0.0.9
    D. 224.0.0.10

18. 在华为路由器上配置 RIP 时，`network 192.168.1.0` 命令的作用是（ ）
    A. 开启 RIP 进程
    B. 在该接口上使能 RIP 并宣告该网段
    C. 设置 RIP 版本
    D. 配置静态路由

19. OSPF 协议中，用于承载链路状态信息的载体是（ ）
    A. Hello 报文
    B. LSA (链路状态通告)
    C. LSU 报文
    D. DD 报文

20. 配置 OSPF 区域 0 时，宣告网段 `192.168.1.0/24` 的正确反掩码是（ ）
    A. 255.255.255.0
    B. 0.0.0.255
    C. 0.0.0.0
    D. 255.255.255.255
