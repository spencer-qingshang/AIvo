# 向往的四级词汇导学

## 1. 课程介绍
- **讲师**: 小杨老师
- **课程**: 向往的四节课 - 词汇部分
- **目标**: 四级冲刺 550 分甚至 600 分以上，而不仅仅是及格。
- **课程体系**: 词汇 -> 语法 -> 练习 -> 全板块精讲 (阅读、听力、写作、翻译) -> 考前救命班。

## 2. 词汇的重要性
- **核心观点**: 英语考试 (四六级、考研、托福等) **80% 的分数取决于词汇能力**。
- **现状分析**: 
    - 四级词汇基本涵盖高考词汇，但由于大一上学期通常无法参加考试，等到大一下或大二参考时，高中词汇量已出现遗忘。
    - 因此，**必须重新系统地背单词**。
- **关于单词书**:
    - 弊端：付出多，收获少；容易只混个眼熟，无法在考试中精准运用。
    - 建议：单词书主要作为查阅工具，而非唯一的背诵来源。

## 3. 核心背单词策略：真题背单词法
这是老师最推荐的高效方法，步骤如下：

1.  **听课打底**: 先听老师讲解核心词汇，确保把老师讲过的词汇背熟。
2.  **限时做题**: 拿到真题 (建议12套)，听讲课前不准做题。听完后，在规定时间内 (如长篇阅读15分钟) 严格按照老师讲的方法做题。
3.  **深度复盘 (核实对错)**: 
    - 不仅要知道正确答案，更要能像老师讲课一样，解释清楚**为什么选A**，以及**为什么B、C、D是错的**。
4.  **全文精读**: 重新通读做过的文章。
5.  **语境查词**: 
    - 遇到生词查出意思，直接**写在真题卷单词旁边**。
    - **切记**: 不要把单词抄到独立的小本子上 (脱离语境容易忘)。
6.  **句中背诵 (最关键)**: 
    - **不要单独背单词** (如：abandon, abandon, 抛弃...)。
    - **要在句子中背单词** (如：Reading "Many Americans have abandoned their dreams during crisis" -> 记忆 abandon)。
    - 总结口诀：**做、查、读、背** (背是灵魂)。

## 4. 辅助记忆技巧 (四大法)

### (1) 谐音法 (发音像中文)
通过发音联想中文含义，适合难记的词：
- **Blush** /bldʃ/ -> "不拉屎" -> 便秘憋得脸色通红 -> **脸红**
- **Flush** /flʌʃ/ -> "福乐时" -> 脸上福乐时 (脸红) / 厕所按钮 -> **脸红 / 冲洗**
- **Splash** /splæʃ/ -> "死不拉屎" -> 严重便秘需吃药 -> **飞溅**

### (2) 联想法 (编故事)
对单词拆解进行趣味联想：
- **Bloom** (B + loom) -> 6100 (Bl近似61) + M (花/美眉) -> 6100朵花 -> **开放 / 盛放 / 繁荣** (e.g., economy is booming)
- **Loom** (100 + M) -> 100米外的美眉 -> 看不清 -> **若隐若现 / 模糊**
- **Gloomy** (G + loomy) -> G (哥哥) 看100米外美眉看不清 -> 心情不好 -> **郁闷的 / 忧郁的**

### (3) 词根词缀法 (科学记忆)
利用词根理解词义逻辑：
- **sp** (词根) -> 联想 "死盼" (往外看)
- **Spy** -> 往外看的人 -> **间谍**
- **Inspect** (in + spect看) -> 往里看 -> **调查 / 研究**
- **Prospect** (pro前 + spect看) -> 往前看 -> **前景 / 前途**
- **Circumspect** (circum圆/环绕 + spect看) -> 环绕着看 -> **小心谨慎的**

### (4) 行近对照法 (易混词比较)
把长得像的单词放在一起记：
- **Moral** (M + oral) -> M (妈妈) + oral (口头的) -> 妈妈口头教导 -> **有道德的**
- **Motor** (M + otor) -> 谐音 "没头" -> **必死的** (与moral对比记忆)
