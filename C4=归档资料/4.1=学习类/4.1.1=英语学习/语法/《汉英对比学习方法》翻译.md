# 学外语就是学翻译

I don't care.无所谓。直译是我不在乎。如果没有汉语译文，仅通过读英语是无法学会这种句子的。只读英语不翻译，会导致输出能力非常低下，翻译能力极其薄弱。

# 学习型翻译

平时所说的翻译都是创造型的翻译，没有答案的翻译。让学生过早这样进行翻译显然不实际。过渡阶段的时候，先让同学们见识汉译英、观察汉译英，而不需要自己动手做。学习英语就是要接受并观察大量的汉英之间的思维转换，并掌握和理解他们。

所谓学习型翻译则是以观察为手段，以比较异同为重点，以记忆英语地道表达为目标。

第一步，英译汉，这步相对简单，学习的成分相应减少。

第二步，汉译英，必须把观察异同的时间增加。

过渡性的话，是不是以后要放弃学习型翻译？

的确，近期目标是记忆英语的地道表达，最终掌握创造型翻译。基于自身水平和客观条件的不同，可能许多人要在很长的一段时间，甚至一辈子都要既做学习型翻译又做创造型翻译，也就是说“现买现卖”。

# 翻译平常化

英语学习中不应把汉译英当成高大上的东西，而是把它当成普通的，基本的工具，这样就能把汉语思维和英语思维相结合，最终达到用英语思维的目标。

# 翻译者平民化

要让英语学习者改变看法，不畏惧翻译，不认为翻译者高不可攀，要让“我也可以做翻译”成为自己的信念。不能光说不练，一定要拿起笔，真刀真枪地干。

我以前总觉得懂英语自然懂翻译，后来发现不是这么回事。

这是一个典型的误区，认为翻译容易，无需大量练习和经验积累，到时自然会，这也是典型的认识误区，无数学生的失败经历便是最好的证明。

# 翻译就是任务

什么才是掌握一篇课文？背下来吗？还是做完课文后面的相关练习？什么程度的理解算是真正理解？把课文背下来算掌握吗？

任何事情，没有目标，最后都会落个一事无成。译学法最好的好处是有个明确的目标，就是把汉语转化成地道的英语，并记在心中。

# 弥补英语环境的缺失

只要在中国学英语，无论什么教学法，语言环境都难以解决。但这不代表学不好英语。

那么不在母语环境学习有什么坏处？

学习语言变成人为和间接的，这样难度会大大增加。

# 翻译技巧

不懂翻译技巧怎么搞翻译？

所谓技巧都是在翻译实践中积累起来的经验总结，如果没有切身体会，身体是不会理解和赞同那些技巧，灵活地、有创造性的运用就更加无从谈起。关于技巧的学习可以看一些英译汉的书。汉译英不需要看，因为有现成汉译英答案，作为初学者认真观察即可。

# 英语学习中的研究与创新

外语专业的研究性学习的确很少。在大学里学的英文，虽然是高等学府，学习的内容却是美国人十一二岁小孩子读的东西。在我国大多数学者不可能像林语堂、胡适那样用英语写东西，就是想写，这样的环境和机会也极少。本质上这种学习是“拿来主义”，一昧的模仿不允许创新，想着把英美人士的表达“据为己有”。

# 翻译是目的也是手段

在课堂上学习游泳，学够了再去游泳池下水练习，当然可以，但是直接去游泳池，让学习的人去水里练习不是更快吗？尽管一开始难一点，但却是最有效的办法。
