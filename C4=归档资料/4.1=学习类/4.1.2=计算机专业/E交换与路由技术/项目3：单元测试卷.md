### 三、 简答题
31. **DHCP 步骤：** (1) Discover (发现)；(2) Offer (提供)；(3) Request (请求)；(4) ACK (确认)。
32. **单臂路由：** 指在路由器物理接口上创建逻辑子接口实现 VLAN 间互访。开启 `arp broadcast enable` 是为了让子接口能接收和处理 ARP 广播报文，从而获取对端 MAC 地址。
33. **路由对比：** 静态路由：配置简单、安全性高、无带宽开销，但不可自动更新，适用于小型稳定网络。动态路由：自动学习、适应拓扑变化，但占用 CPU 和带宽，适用于大型复杂网络。
34. **RIP 区别：** (1) v1 是有类别路由协议，v2 是无类别；(2) v1 广播更新，v2 组播更新（224.0.0.9）；(3) v2 支持 VLSM 和路由认证。
35. **OSPF 区域：** 划分 Area 0 是为了防止路由环路并作为跨区域通信的中枢。要求：所有非骨干区域必须直接与骨干区域相连。

### 四、 应用题
36. 
(1) 配置命令：
```shell
ip route-static 0.0.0.0 0.0.0.0 10.1.1.2
ip route-static 0.0.0.0 0.0.0.0 20.1.1.2 preference 80
```
(2) 判断逻辑：路由器根据优先级（Preference）判断。数值越小优先级越高。正常情况下，优先级 60 的路由进入路由表；当 10.1.1.2 链路失效时，优先级 80 的备份路由才会加入路由表。

37. 
(1) R1 配置命令：
```shell
ospf 1 router-id 1.1.1.1
 area 0
  network 10.0.12.0 0.0.0.255
```
(2) 反掩码作用：用于指定需要参与 OSPF 进程的接口范围（0 表示匹配，1 表示忽略）。宣告主机地址的反掩码为：`0.0.0.0`。
