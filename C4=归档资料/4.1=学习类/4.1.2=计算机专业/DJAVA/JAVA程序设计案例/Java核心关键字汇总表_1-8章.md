# Java核心关键字与重要语法汇总表 (1-8章)

本表格汇总了您学习笔记中第一章至第八章涵盖的核心关键字、重要类名和方法，以便于您快速查阅和背诵。

---

### 1. 基础语法与流程控制

| 关键字/语法     | 所属章节 | 核心作用/解释                                                |
| :-------------- | :------- | :----------------------------------------------------------- |
| `class`         | 第一章   | 定义一个类。                                                 |
| `public`        | 第一/五章 | 访问修饰符，表示公开，任何地方都可访问。                     |
| `static`        | 第一/五章 | 静态修饰符，表示成员属于类，被所有对象共享。                 |
| `void`          | 第一/三章 | 返回值类型，表示方法不返回任何值。                             |
| `main`          | 第一章   | 特殊方法名，是Java程序的入口点。                             |
| `new`           | 第二/五章 | 创建对象（实例）的关键字。                                   |
| `import`        | 第二章   | 导入其他包中的类。                                           |
| `final`         | 第二章   | 修饰符，修饰的变量变为常量（只能赋值一次），修饰的类不能被继承，修饰的方法不能被重写。 |
| `if` / `else`   | 第三章   | 分支结构关键字，用于条件判断。                               |
| `switch` / `case` / `break` / `default` | 第三章 | 分支结构关键字，用于固定值匹配。`break`用于跳出，`default`用于处理未匹配情况。 |
| `for` / `while` / `do-while` | 第三章 | 循环结构关键字，用于重复执行代码。                         |
| `continue`      | 第三章   | 跳过本次循环，进入下一次循环。                               |
| `return`        | 第三章   | 结束方法的执行，并可返回一个值。                             |

### 2. 基本数据类型

| 关键字    | 所属章节 | 核心作用/解释                   |
| :-------- | :------- | :------------------------------ |
| `byte`    | 第二章   | 8位整型，范围-128到127。        |
| `short`   | 第二章   | 16位整型。                      |
| `int`     | 第二章   | 32位整型，是整型的默认类型。    |
| `long`    | 第二章   | 64位整型，数值后需加`L`或`l`。   |
| `float`   | 第二章   | 32位单精度浮点型，数值后需加`F`或`f`。 |
| `double`  | 第二章   | 64位双精度浮点型，是浮点型的默认类型。 |
| `char`    | 第二章   | 16位Unicode字符型，可存单个汉字。 |
| `boolean` | 第二章   | 逻辑类型，只有`true`和`false`两个值。 |

### 3. 面向对象与继承

| 关键字/语法     | 所属章节 | 核心作用/解释                                                |
| :-------------- | :------- | :----------------------------------------------------------- |
| `private`       | 第五章   | 访问修饰符，表示私有，只能在本类中访问，是封装的核心。     |
| `protected`     | 第五章   | 访问修饰符，表示受保护，本包及外包子类可访问。             |
| `this`          | 第五章   | 代表当前对象的引用，用于区分同名变量或调用本类构造器。     |
| `extends`       | 第五章   | 继承关键字，一个类继承另一个类。                             |
| `super`         | 第五章   | 代表父类对象的引用，用于调用父类的构造器或成员。           |
| `@Override`     | 第五章   | 注解，表示该方法是重写父类的方法，可让编译器帮助检查。     |
| `instanceof`    | 第五章   | 运算符，判断一个对象是否是某个类（或其子类）的实例。       |
| `abstract`      | 第六章   | 抽象修饰符，修饰类（抽象类）或方法（抽象方法）。           |
| `interface`     | 第六章   | 定义一个接口。                                               |
| `implements`    | 第六章   | 实现关键字，一个类实现一个或多个接口。                     |
| `Object`        | 第七章   | 所有类的根类（祖宗类）。                                     |
| `toString()`    | 第七章   | `Object`类的方法，返回对象的字符串表示，常被重写。           |
| `equals()`      | 第七章   | `Object`类的方法，比较两个对象是否相等，常被重写以比较内容。 |

### 4. 异常处理

| 关键字    | 所属章节 | 核心作用/解释                                                |
| :-------- | :------- | :----------------------------------------------------------- |
| `try`     | 第六章   | 包裹可能发生异常的代码块。                                   |
| `catch`   | 第六章   | 捕获并处理在`try`块中发生的异常。                            |
| `finally` | 第六章   | 无论是否发生异常，总会执行的代码块，常用于释放资源。       |
| `throw`   | 第六章   | 在**方法体内部**手动创建并抛出一个异常对象。                 |
| `throws`  | 第六章   | 在**方法声明处**声明该方法可能抛出的异常类型。               |

### 5. 常用类与集合框架

| 关键字/语法       | 所属章节 | 核心作用/解释                                                |
| :---------------- | :------- | :----------------------------------------------------------- |
| `String`          | 第七章   | **不可变**的字符串类。                                       |
| `StringBuffer`    | 第七章   | **可变**的、**线程安全**的字符串类，适合频繁拼接。           |
| `Integer`等包装类 | 第七章   | 将基本数据类型封装成对象，使其能存入集合。                 |
| `Arrays`          | 第四章   | 操作数组的工具类，提供`sort()`, `toString()`等静态方法。     |
| `Collection`      | 第八章   | 单列集合的根接口。                                           |
| `List`            | 第八章   | `Collection`的子接口，特点：**有序、可重复**。                 |
| `Set`             | 第八章   | `Collection`的子接口，特点：**无序、不可重复**。               |
| `Map`             | 第八章   | 存储**键值对**的顶层接口，特点：**键唯一、值可重复**。       |
| `ArrayList`       | 第八章   | `List`接口的常用实现类，底层是数组，查询快，增删慢。         |
| `HashSet`         | 第八章   | `Set`接口的常用实现类，底层是哈希表，无序。                  |
| `HashMap`         | 第八章   | `Map`接口的常用实现类，底层是哈希表，无序。                  |
| `Iterator`        | 第八章   | 迭代器接口，用于遍历集合的标准方式。                         |
| `Collections`     | 第八章   | 操作`Collection`和`Map`的**工具类**，提供`sort()`, `shuffle()`等静态方法。 |
| `Comparable`      | 第八章   | **内部比较器**接口，让类自身具备默认排序能力。               |
| `Comparator`      | 第八章   | **外部比较器**接口，用于定义临时的、定制化的排序规则。     |
