# 英汉词汇不对应

单词量的大小不能光看认得多少，而是会用多少。能用就是在说英语、写英语和汉译英时能想起来，并知道该不该用，怎样用。一个单词的意思会有很多种，就算是apple也会被当成比喻成别的意思。

# 概念意思

英汉词汇的对比是可以从两个方面切入，一是概念意义的对比，二是关联意义的对比。词典给的定义就是概念意义。很多词都是多义词，会有多个概念意义。

英美词汇在理解上偏差会很大，甚至背道而驰。所以查词典的时候要查英英词典，因为英汉词典很多容易有错。

# 次要意义

语言是及其细微、复杂的，只要有用的意义就要记住。例如中文的黄色可以代表色情，英语中就没有这个意思。

# 比喻意义

英语词典表为fig.或figurative，例如中文里的望子成龙，龙就是比喻意义，如果英语中没有这个比喻意义，他们的文化不同，表达的意思就变了。

过渡概括是语言学习中很常见的现象，例如car、book、apple等词加上s就能表示复数，便能概括出英语名词加-s可以构成复数的规律，如果把man,woman等也概括进去就叫做过渡概括。

比如上的过渡概括，犯这种错误是因为不了解语言之间的差异。

# 文体意义

关联意义主要包括文体意义、感情意义、搭配意义和联想意义。

文体意义指的是词应用于某种场合的特性。

怎么样主义到词的文体意义呢？

依靠词典，查他们的意思。

# 感情意义

每个词都有褒义和贬义。有的词既有褒义也有贬义。一定要看清楚有没有approving, disapproving；apprec,derog这样的标识

# 搭配意义

指两个或两个以上的词一起使用时的习惯配合。

这种搭配起来的词意思会完全不同，在中文和英语都是一样，所以背单词不可孤立记单词。

# 联想意义

联想意义在词典上很难学习，因为这是因语言而异、因文化而异、因国家而异、因民族而异、因时间而异、因地域而异，甚至因人而异。

次内意义，一个词可能会关乎历史，民族，会让人有极其憎恶的想法。例如红色在中国是代表喜庆，在西方文化里是代表血腥。

# 词汇空缺

比如包子、棺材，对方语言里没有这种词汇。空缺的转化可以称得上是艺术。

# 词汇学习的最终目的是输出

第一，单词要在句子里学，在单词表里背是不可取的。不要背单词，要背句子，既可以了解词义，还知道了用法和搭配，还学会了完整的句子表达

第二，一些具体意思明确的词，意义单纯的词可以直接背。

要三，学习英语词汇要把输出放在心上。

第四，学习、研究和记忆机会的时候要尽多地从汉语朝英语方向，也就是倒着学、倒着背。

第五，不同的英汉对应词之间的情况大同小异，小同大异，要不停找出异同然后运用。
