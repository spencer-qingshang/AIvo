在考卷中，如果问到**动态 NAT（Dynamic NAT）**的作用和优点，阅卷老师希望看到的是你拿它去和**“静态 NAT”**做对比。

### 一、动态 NAT 的主要作用

它的核心作用可以概括为：**“先到先得的公共 IP 池分配”**。

- **动态转换：** 它将内部网络的私有 IP 地址，动态地映射到路由器预先配置好的一个“公网 IP 地址池（Address Pool）”中。
    
- **按需分配，用完回收：** 当内网某台电脑需要上网时，路由器从池子里挑一个处于“空闲”状态的公网 IP 借给它。等这台电脑断网下线后，这个公网 IP 就会被**回收**到池子里，留给下一个需要上网的人用。
    
- **注意（考试重点）：** 它**只转换 IP 地址，不转换端口号（No-PAT）**。因此在同一时刻，池子里有几个公网 IP，内网就只能有几台电脑同时上网。
    

---

### 二、动态 NAT 的优点（主要是对比静态 NAT 而言）

虽然它在现网中已经被强大的 NAPT（端口多路复用）取代，但在理论考试中，它相比于最基础的静态 NAT，拥有以下三大优点：

**1. 提高了公网 IP 的利用率（最核心优点）**

- **对比静态 NAT：** 静态 NAT 是“占着茅坑不拉屎”，哪怕内网电脑关机了，那个公网 IP 别人也用不了。
    
- **动态 NAT 的优势：** 实现了公网地址的“轮流坐庄”。比如公司有 100 名员工，但购买了 20 个公网 IP。只要同一时刻上网的人数不超过 20 人，大家就可以通过这个池子轮流上网，这比买 100 个静态 IP 省钱得多。
    

**2. 增强了内网的安全性**

- **隐藏拓扑：** 和所有的 NAT 技术一样，它对外隐藏了内网真实的 IP 架构，外部网络看到的只是地址池里的公网 IP。
    
- **不可预测性：** 因为分配是动态的，某台内网电脑今天用的公网 IP 是 `200.1.1.10`，明天可能就变成了 `200.1.1.12`。外部的黑客无法通过固定的公网 IP 去长期锁定并攻击某台特定的内网主机。同时，由于它是单向触发的（只能内网主动发往外网），外网无法主动通过动态 NAT 穿越到内网。
    

**3. 配置相对简便，易于批量管理**

- 不需要像静态 NAT 那样，由网络工程师手工去一条条地写“IP 1 绑定 IP A，IP 2 绑定 IP B”的死板规则。只需要划定一个内网范围（用 ACL），再画一个外网池子，路由器就会自动搞定它们之间的分配和回收逻辑。
