# 最让人恼是口语

口语说的不好会很叫人没面子，许多同学在阅读、语法和应负考试等方面都不错，可就是口语不行，一张嘴就掉链子。让外国人见了笑话。

# 思想与口语

一个人不善于表达，可能是因为他没有思想，比如说对某个问题的观点、想法或看法。如果在汉语中都表达不出来，更别说用英语了。现在译学法可以解决这个问题，因为思想就包含在汉语句子里，只要努力学习和练习将汉语转化成英语就可以了饿。这不但是练习的有效手段，还能节省大量精力和时间，并在转化过程，吸收和学习别人的观点和思想。

# 简单也需翻译和对比

时间会治愈创伤

将这句话翻译成英文

The time can cure the wound.

英语中的表达并没有“会”的对应词，但是却有一般现在时态来告诉你，用来表示普遍真理，这时便可以译为“会......”

Time cures all things.

# 不用没学过的英语

使用英语表达时一定要学过或阅读过的句子，结构、表达式和词组。也可以在其基础上改造或仿造。注意，英语译学法的最基本原则就是模仿式学习。

侧重点在于强调模仿式学习，即学了什么，就记什么，就说什么。

# 英语口语课

现代口语课仅仅只在简单的东西上一直重复，并不能让你学到真正的东西，效率很低下。而且请的是外教当老师，这样下来就不能用英语译学法，尽管他们有着标准的发音和英美文化，但并不能解决中国学生脑子里一直想着中文。

# 英语俚语

英语俚语是要学习的，因为这时real English，即便在国内英语学得好，到了外国也会听不懂美国人说的是什么，但要注意场合的使用。

# 注意口语语体

把名著里的单词用在现实里，老外会听不懂你说的什么，还会觉得你很装。
