# 英语学习知识库 (New Structure)

> 欢迎使用重构后的英语学习笔记体系！

## 📂 目录说明
- **[00=仪表盘](./00=仪表盘/英语学习主页.md)**：您的中央入口，显示当前任务、待复习单词和学习统计。
- **[10=学习素材](./10=学习素材/)**：存放剧本、真题原文。
- **[20=知识库](./20=知识库/)**：存放原子化的单词卡片和语法点。
- **[30=复盘](./30=复盘/)**：存放每日总结和错题记录。
- **[40=考试](./40=考试/)**：存放考试大纲。
- **[模板](./模板/)**：创建新笔记时使用的标准 Markdown 模板。

## 🚀 核心功能
1.  **自动化汇总**：通过 `20=知识库` 里的 `生词本.md` 和 `30=复盘` 里的 `错题库.md` 自动汇总所有记录。
2.  **复习提醒**：仪表盘会根据掌握度自动提醒您需要复习的内容。
3.  **原子化存储**：一个单词一个文件，方便以后进行双向链接。

## 🛠️ 维护建议
- 使用 `模板` 目录下的文件创建新笔记。
- 确保 YAML 区域（文件开头 `---` 之间的部分）填写正确，否则自动化视图无法抓取。
