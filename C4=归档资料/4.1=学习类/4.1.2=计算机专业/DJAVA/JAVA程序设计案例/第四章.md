# 1. 一维数组

## 识记：

### 数组的概念；

- **概念**：数组是一种**容器**，用来存储**同种数据类型**的多个值。

### 数组的定义方法。

- **特点（必考）**：

- **定长**：数组一旦创建，**长度不可改变**。

- **同质**：所有元素的类型必须相同（全是 int 或全是 String）。

- **引用类型**：数组本身属于引用数据类型（即使存的是基本数据）。

- **定义格式**：

- **推荐写法**：数据类型[] 数组名; （如 int[] arr;）

- *C语言写法*：数据类型 数组名[]; （Java 也支持，但不推荐）。

## 领会：

### 数组的作用；

- **索引/下标 (Index)**：

- 数组中的每个元素都有一个编号，**从 0 开始**。

- 最大下标 = **数组长度 - 1**。

- **访问方式**：通过 数组名[索引] 来获取或修改元素。

- **长度获取**：使用 **数组名.length** 属性。

- *超级考点*：数组的 length 是**属性**（没有括号）；字符串的 length() 是**方法**（有括号）。

### 数组的相关操作。

**A. 数组初始化 (Initialization)**

- **静态初始化**（知道具体值时用）：

- int[] arr = {10, 20, 30};

- *完整写法*：int[] arr = new int[]{10, 20, 30};

- **动态初始化**（只知道长度时用）：

- int[] arr = new int[5]; （创建了5个位置）。

- *默认值规则*：int 是 0，double 是 0.0，boolean 是 false，引用类型是 null。

**B. 基本操作（增删改查）**

- **修改/赋值**：arr[0] = 100;

- **获取**：int x = arr[0];

- **遍历 (Traversal)**（必背代码）：

```java
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

**C. 常见算法**

- **查找**：遍历数组，用 if (arr[i] == target) 判断。

- **最值**：假设第一个是最大值 max = arr[0]，然后遍历比较。

4. 应用：Arrays 工具类的使用 (The Arrays Class)

java.util.Arrays

- **打印数组**：**Arrays.toString(arr)**

- *考点*：直接 System.out.println(arr) 会打印出内存地址（乱码），必须用 toString 才能看到内容 [1, 2, 3]。

- **数组排序**：**Arrays.sort(arr)**

- *效果*：默认**升序**排列（从小到大）。

- **二分查找**：Arrays.binarySearch(arr, key)

- *前提*：数组必须**已经排好序**。

# 2. 二维数组

## 识记：

### 二维数组概念。

- **本质**：**数组的数组**。二维数组中的每一个元素，本身又是一个一维数组。

- **形象理解**：像 Excel 表格、围棋棋盘、电影院座位表。

- **结构特征**：由**行 (Row)** 和 **列 (Column)** 组成。

## 领会：

### 二维数组作用、应用场景及相关操作。

- **应用场景**：

- 存储**矩阵**数据（数学运算）。

- 游戏开发（如**地图**、迷宫、棋盘坐标）。

- 存储表格数据（学生成绩表：行是学生，列是科目）。

- **访问方式**：需要两个下标。

- 格式：**数组名[行索引][列索引]**

- *注意*：下标都是从 **0** 开始的。

- **长度概念（易错点）**：

- **arr.length**：表示**有多少行**（二维数组的长度）。

- **arr[i].length**：表示**第 i 行有多少列**（里面那个一维数组的长度）。

## 应用：

### 二维数组定义、**初始化**；

**A. 定义与初始化**

- **动态初始化**（只指定大小，默认值为 0）：

- 格式：数据类型[][] 数组名 = new 数据类型[行数][列数];

- *例*：int[][] arr = new int[3][4];

- *解释*：创建了一个 3 行 4 列的数组。

- **静态初始化**（直接给值）：

- 格式：数据类型[][] 数组名 = {{...}, {...}, ...};

- *例*：

```java
int[][] arr = {
    {1, 2, 3},   // 第0行
    {4, 5, 6},   // 第1行
    {7, 8, 9}    // 第2行
};
```

### 二维数组遍历。

使用

```java
int[][] arr = {{1, 2}, {3, 4}, {5, 6}};

// 外层循环：遍历每一行 (i < arr.length)
for (int i = 0; i < arr.length; i++) {
    // 内层循环：遍历该行中的每一列 (j < arr[i].length)
    for (int j = 0; j < arr[i].length; j++) {
        System.out.print(arr[i][j] + " "); // 打印元素
    }
    System.out.println(); // 每一行打完，换个行
}
```
