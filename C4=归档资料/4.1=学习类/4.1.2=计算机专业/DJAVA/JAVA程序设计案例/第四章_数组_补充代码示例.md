### 1. 一维数组 (One-Dimensional Array) 示例

这个例子涵盖了一维数组的创建、初始化、遍历和基本算法（求最大值）。

```java
public class OneDimArrayExample {
    public static void main(String[] args) {
        // --- 1. 数组的初始化 ---
        // 静态初始化：在定义时就指定了所有元素
        int[] scores = {85, 92, 78, 100, 63};
        
        // 动态初始化：只指定数组长度，元素会被赋予默认值 (int的默认值是0)
        String[] names = new String[5];
        names[0] = "张三";
        names[1] = "李四";
        
        // --- 2. 访问和修改元素 ---
        System.out.println("第一个学生的分数是: " + scores[0]); // 输出: 85
        scores[0] = 88; // 修改第一个元素的值
        System.out.println("修改后第一个学生的分数是: " + scores[0]); // 输出: 88
        
        // --- 3. 遍历数组 ---
        System.out.println("\n--- 所有学生的分数 ---");
        for (int i = 0; i < scores.length; i++) {
            System.out.print(scores[i] + " "); // 输出: 88 92 78 100 63 
        }
        
        // --- 4. 基本算法：求最大值 ---
        int maxScore = scores[0]; // 假设第一个元素是最大的
        for (int i = 1; i < scores.length; i++) {
            if (scores[i] > maxScore) {
                maxScore = scores[i]; // 如果发现更大的，就更新maxScore
            }
        }
        System.out.println("\n最高分是: " + maxScore); // 输出: 100
    }
}
```

### 2. 二维数组 (Two-Dimensional Array) 示例

二维数组本质上是“数组的数组”，常用于表示表格或矩阵。

```java
public class TwoDimArrayExample {
    public static void main(String[] args) {
        // 静态初始化一个 3x2 的二维数组
        int[][] matrix = {
            {1, 2},
            {3, 4},
            {5, 6}
        };
        
        // --- 1. 访问元素 ---
        // 访问第1行、第0列的元素 (行和列的索引都从0开始)
        System.out.println("matrix[1][0] 的值是: " + matrix[1][0]); // 输出: 3
        
        // --- 2. 获取行数和列数 ---
        System.out.println("二维数组的行数: " + matrix.length); // 输出: 3
        System.out.println("第0行的列数: " + matrix[0].length); // 输出: 2
        
        // --- 3. 遍历二维数组 ---
        System.out.println("\n--- 遍历二维数组 ---");
        // 外层循环控制行
        for (int i = 0; i < matrix.length; i++) {
            // 内层循环控制列
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + "\t"); // 使用\t制表符对齐
            }
            System.out.println(); // 每打印完一行后换行
        }
        /*
         * 输出结果:
         * 1\t2	
         * 3\t4	
         * 5\t6	
         */
    }
}
```

### 3. `Arrays` 工具类示例

`java.util.Arrays` 类提供了很多方便的静态方法来操作数组。

```java
import java.util.Arrays;

public class ArraysClassExample {
    public static void main(String[] args) {
        int[] numbers = {50, 20, 90, 10, 80};
        
        // 1. 打印数组内容 (toString)
        // 如果直接打印数组对象，会得到内存地址，如 [I@1b6d3586
        System.out.println("排序前: " + Arrays.toString(numbers)); // 输出: [50, 20, 90, 10, 80]
        
        // 2. 排序 (sort) - 默认升序
        Arrays.sort(numbers);
        System.out.println("排序后: " + Arrays.toString(numbers)); // 输出: [10, 20, 50, 80, 90]
        
        // 3. 二分查找 (binarySearch) - 前提：数组必须已排序！
        int target = 90;
        int index = Arrays.binarySearch(numbers, target);
        System.out.println("元素 " + target + " 的索引是: " + index); // 输出: 4
        
        int targetNotFound = 60;
        int indexNotFound = Arrays.binarySearch(numbers, targetNotFound);
        // 如果找不到，会返回一个负数。
        System.out.println("查找不存在的元素 60: " + indexNotFound); // 输出: -4 (一个负数)
    }
}
```
