# Implementation Plan: 2026年春季攻坚

本计划分为三个阶段：PETS-3 冲刺、自考基础夯实、自考全真模拟冲刺。

---

## Phase 1: PETS-3 决战前夕 (2月-3月考前)
**目标：** 全力备战英语，确保听力和阅读不失分。同时维持“交换与路由”的手感。

- [ ] Task: **[PETS-3]** 建立 PETS-3 专项复习笔记结构
    - [ ] 在 `C4` 中建立 PETS-3 错题本和高频词汇表。
    - [ ] 整理 10 篇通用写作模板（信函、通知、议论文）。
- [ ] Task: **[PETS-3]** 听力特训周
    - [ ] 每天早晚各 30 分钟磨耳朵（使用历年真题音频）。
    - [ ] 完成 5 套完整听力真题，并进行精听复盘。
- [ ] Task: **[PETS-3]** 阅读与写作限时训练
    - [ ] 每周末进行一次全真模拟（笔试部分），严格控制时间（120分钟）。
    - [ ] 批改并记录错题。
- [ ] Task: **[交换路由]** 维持热度
    - [ ] 每周至少安排 2 小时 eNSP 实验（如 VLAN 配置、静态路由），避免遗忘命令。
- [ ] Task: Conductor - User Manual Verification 'Phase 1' (Protocol in workflow.md)

---

## Phase 2: 自考公共课突击 (3月考后-4月初)
**目标：** 在 PETS-3 考完后，无缝切换到高强度的背诵模式。重点攻克马原和习思想。

- [ ] Task: **[马原/习思想]** 建立知识骨架
    - [ ] 整理《马原》三大板块（唯物论、辩证法、政经）的思维导图。
    - [ ] 整理《习思想》的最新教材目录大纲。
- [ ] Task: **[马原/习思想]** 核心考点背诵
    - [ ] 制定背诵计划，每天早晚背诵简答题/论述题核心点。
    - [ ] 利用碎片时间刷选择题（建议使用刷题 App）。
- [ ] Task: **[大学语文]** 文言文与作文
    - [ ] 复习教材规定的必读古文篇目（重点字词解释）。
    - [ ] 准备 3-5 个万能作文素材（立志、爱国、传统文化等）。
- [ ] Task: Conductor - User Manual Verification 'Phase 2' (Protocol in workflow.md)

---

## Phase 3: 全科目冲刺与模考 (4月考前两周)
**目标：** 查漏补缺，适应考试节奏。

- [ ] Task: **[交换路由]** 理论题突击
    - [ ] 集中刷历年真题中的选择题和填空题。
    - [ ] 复习简答题（如 STP 选举过程、ARP 工作原理）。
- [ ] Task: **[全科]** 真题模拟
    - [ ] 马原：完成近 3 年真题，重点看大题押题。
    - [ ] 习思想：关注时政结合点。
    - [ ] 语文：写一篇完整作文练手。
- [ ] Task: **[考前准备]**
    - [ ] 打印准考证，确认考场路线。
    - [ ] 准备考试文具（2B铅笔、黑色签字笔）。
- [ ] Task: Conductor - User Manual Verification 'Phase 3' (Protocol in workflow.md)
