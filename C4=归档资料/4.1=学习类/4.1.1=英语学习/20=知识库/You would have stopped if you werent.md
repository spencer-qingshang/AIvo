# You would have stopped... if you weren't...

> [!abstract] 结构解析
> 这是一个典型的**虚拟语气**句型，用于对过去已经发生的事情进行“假设性吐槽”。

---

## 1. 语法结构
- **主句：** `Subject + would have + done` (表示本可以做到某事，但实际没做到)。
- **从句：** `If + subject + were not doing` (表示如果当时没在做某事，但实际正在做)。

## 2. 剧集语境
- **原句：** `You would have stopped that goal if you weren't staring at that little girl.`
- **翻译：** 如果你刚才没在盯着那个小女孩看，你本可以挡住那个球的。

## 3. 学习心得
当你想要表达“要不是因为你刚才在...，你本来可以...”这种责备语气时，这个句型是最高级的选择。
