# 交换与路由技术 错题本

## 单选题

### 1. 802.1Q 标签长度
- **题目**：一个标准的 802.1Q 帧在原始以太网帧的基础上，增加了几个字节的 Tag 标签字段？
- **答案**：B. 4 字节
- **知识点**：802.1Q Tag 长度为 4 字节，包含 TPID (2B) 和 TCI (2B)。VLAN ID 位于 TCI 中，占 12 位。
- **记录日期**：2026-02-10

### 2. 跨交换机 VLAN 通信端口类型
- **题目**：若要实现跨越两台交换机 subterranean 的相同 VLAN 内部主机进行通信，连接两台交换机之间的链路端口类型通常应配置为？
- **答案**：B. Trunk
- **知识点**：Trunk 端口用于交换机级联链路，允许多个 VLAN 流量带标签（Tag）通过。Access 端口仅支持单一 VLAN，通常连接终端。
- **记录日期**：2026-02-10

### 3. STP 桥 ID (BID) 组成
- **题目**：在 STP 中，桥 ID (Bridge ID) 是由哪两部分组成的？
- **答案**：B. 桥优先级 和 桥地址 (MAC 地址)
- **知识点**：BID = 优先级 (16位) + MAC地址 (48位)。优先级默认 32768，步长 4096。选举根桥时 BID 越小越优先。
- **所属章节**：项目2 交换技术
- **记录日期**：2026-02-10

### 4. RSTP 端口状态
- **题目**：在 RSTP 中，处于哪种状态的端口既不转发用户流量，也不学习 MAC 地址？
- **答案**：A. Discarding
- **知识点**：RSTP 将状态精简为 Discarding、Learning 和 Forwarding。Discarding 状态整合了 STP 的禁用、阻塞和监听状态。
