# Track: 补充项目6、7大纲及生成园区网实操手册 (spec.md)

## 1. 概述
本项目旨在完善 "项目6 构建安全的园区网络.md" 和 "项目7 园区网络综合实训.md" 的大纲内容。通过深度补充“流程原理级”的大白话讲解，并生成对应的 4 份实操手册（项目6拆分3份，项目7一份），帮助用户完成从基础安全到综合组网的进阶学习。

## 2. 功能要求

### 2.1 补充理论大纲
*   **输入**：`项目6 构建安全的园区网络.md`, `项目7  园区网络综合实训.md`。
*   **内容补充要求**：
    *   **项目 6**：用大白话解释 ACL（过滤器）、NAT（身份证翻译官）、防火墙（海关与安检）等核心逻辑。
    *   **项目 7**：解释如何将之前学的零散知识（VLAN, OSPF, WLAN, 安全）拼装成一个完整的骨架。
*   **风格**：保持生动有趣的类比，增加“为什么这么做”的解释。

### 2.2 生成实操手册集
所有手册均需包含：**拓扑图描述、规划表、避坑指南、分步配置（带大白话注释）、验证方法**。

#### 项目 6 拆分手册：
1.  **项目6-1：《交换机接入安全实战》**
    *   重点：端口安全（只认熟脸）、AAA 认证、Telnet 远程管理。
2.  **项目6-2：《路由器流量控制与NAT实战》**
    *   重点：基本/高级 ACL（防火防盗防闺蜜）、NAT（私网转公网）。
3.  **项目6-3：《防火墙基础实战》**
    *   重点：安全区域（信任区、非信任区）、安全策略。

#### 项目 7 综合手册：
4.  **项目7：《中型企业网综合实训手册》**
    *   架构：二层接入 + 三层核心（网关+DHCP） + 出口路由（NAT） + 无线管理（AC/AP）。
    *   目标：全网互通 + 安全访问控制。
