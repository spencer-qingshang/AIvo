### 去心邻域的概念

有时候，我们只考虑点 $a$ 邻近的点，**不考虑点 $a$**，即考虑点集 $\{x \mid a-\delta < x < a \text{ 且 } a < x < a+\delta\}$。

我们称这个点集为点 $a$ 的 **去心邻域**，记为 $\mathring{U}(a, \delta)$，即：

$$
\mathring{U}(a, \delta) = \{x \mid a-\delta < x < a \text{ 且 } a < x < a+\delta\}
$$

*注：也可以利用绝对值等价表示为 $0 < |x - a| < \delta$*

### 三、绝对值 ⭐

设 $x$ 是一实数，用 $|x|$ 记 $x$ 的绝对值，其定义如下：

$$
|x| = \begin{cases} 
x, & x \ge 0 \\
-x, & x < 0 
\end{cases}
$$

**几何意义：**
* $|x|$ 的几何意义是点 $x$ 到原点的距离。
* 显然，$|x - y|$ 表示点 $x$ 与点 $y$ 之间的距离。

**例：**
$|1.52| = 1.52$
$|-2.50| = 2.50$
$|0| = 0$

### 绝对值的性质 (这部分证明需要另外刷题，以及自己证明一遍)

绝对值有下列性质：设 $x, y$ 是实数，则：

1. $|x| \geqslant 0$，当且仅当 $x=0$ 时才有 $|x|=0$；
2. $|-x| = |x|$；（往左走五步和往右走五步，距离是一样的，所以是相等的）
3. $|xy| = |x||y|$；$|x^2 |=x^2(因为平方后肯定是正数，所以绝对值符号就没用了)$
4. $a>0$ 时，$|x|<a$ 当且仅当 $-a<x<a$；（这个是领域，x 肯定是在-a 到 a 的范围内）
5. $-|x| \leqslant x \leqslant |x|$；（对于任何一个数，要么是正，要么是负，要么是 0，最大不会超过自己的绝对值，当是整数的时候，最小不会超过自己绝对值的相反数）
6. $|x+y| \leqslant |x|+|y|$；
7. $|x-y| \geqslant ||x|-|y|| \geqslant |x|-|y|$。

$|x|\geqslant0$，当且仅当 $x=0$ 时才有 $|x|=0$;
（可以翻译 x 为到原点的距离，距离不可能是负数）
