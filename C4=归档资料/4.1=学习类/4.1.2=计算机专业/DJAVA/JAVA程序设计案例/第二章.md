# 1. java关键字与标识符

## 识记：

### java主要关键字；

- **定义**：Java 语言中被赋予了**特殊含义**的单词。

- **书写规则（必考）**：所有的关键字**全部都是小写**。例如：if 是关键字，但 If、IF 都不是。

- **使用限制**：关键字**不能**用作变量名、方法名或类名。

- **goto**** 和 ****const**：这两个词在 Java 中被保留（Reserved Words），但目前**没有任何功能**（未被使用）。

- **数据类型**：int, double, boolean, char, void 等。

- **流程控制**：if, else, switch, for, while, break, return 等。

- **修饰符**：public, private, static, final, abstract 等。

- **类与对象**：class, extends, new, this, super 等。

- **异常处理**：try, catch, finally, throw, throws。

### java标识符语法规范；

- **组成元素**：只能包含 **字母**、**数字**、**下划线 (****_****)**、**美元符号 (****$****)**。

- **开头限制**：**绝不能以数字开头**。*例子*：1abc 是错的，abc1 是对的。

- **避讳**：**不能使用关键字**（如 class、public、int 等）。

- **特性**：**严格区分大小写**（myVar 和 MyVar 是两个不同的名字）。

- **其他**：没有长度限制；支持 Unicode（理论上可以用中文命名，但不推荐）。

### 标识符命名风格。

- **原则**：**见名知意**（一看就知道是干嘛的，如 studentName）。

- **包名 (Package)**：**全小写**（如 com.baidu）。

- **类名/接口名 (Class)**：**大驼峰**（首字母大写，如 ArrayList）。

- **变量/方法名 (Variable/Method)**：**小驼峰**（首字母小写，后面单词首字母大写，如 getMaxScore）。

- **常量名 (Constant)**：**全大写**，单词间用**下划线**隔开（如 MAX_VALUE）。

# 2. 常量与变量

## 识记：

### 常量的表示；

- **概念**：在程序运行过程中，**值固定不变**的量。

- **作用**：用于表示不能修改的数据（如圆周率 PI）。

- **关键字**：必须使用 **final** 进行修饰。

- **别名**：声明常量也常被称为“**final 变量**”。

- **赋值规则**：在整个程序中**只能被赋值一次**。*考点*：如果尝试对 final 变量进行第二次赋值，系统会报**编译错误**。*注意*：可以先声明，后赋值（但仍只能赋一次）。

- **语法格式**：final 数据类型 常量名 = 值;

- **命名规范**：虽不是强制，但习惯上常量名使用**全大写字母**。如果单词多，用下划线隔开（如 MAX_VALUE），以便于区分变量。

- **字面值常量**

- **分类**：

- **整型**：123

- **浮点型**：4.56

- **字符型**：'A' (单引号)

- **字符串型**："Hello" (双引号)

- **布尔型**：true / false

- **空常量**：null

### 变量的概念。

- **定义**：在程序运行过程中，其**值可以发生改变**的量。

- **本质**：变量是内存中的一小块区域。

- **三要素（必考）**：

- **类型** (Type)：决定了能存什么样的数据（如 int, double）。

- **变量名** (Name)：用来标识这块内存的名字。

- **值** (Value)：这里面具体存的数据。

## 领会：

### 变量的作用。

- **存储数据**：相当于一个**容器**（或盒子），用来存放各种数据（如年龄、余额、名字）。

- **分配内存**：声明一个变量，系统就会在内存中**配置一块空间**给它。

- *考点*：不管变量的值怎么变，这块空间的**地址**是不变的（房子还是那个房子，里面的租客换了）。

# 3. 基本数据类型

## 识记：

### 数据类型作用；

- **定义**：Java 将数据按性质进行分类。

- **四大作用**：数据类型决定了数据的**性质**、**取值范围**、**存储方式**以及能进行的**运算**。

### 数据类型分类与表示。

- **两大阵营**：Java 数据类型分为 **基本数据类型** 和 **引用数据类型**。

- **8 种基本数据类型 (The 8 Primitives - 必考)**

- **整型 (4种)**：用于存整数。**byte** (最小)、**short** (短整)、**int** (最常用)、**long** (长整)。

- **浮点型 (2种)**：用于存小数。**float** (单精度)、**double** (双精度，更准)。

- **字符型 (1种)**：用于存单个字符。**char** (如 'A')。

- **布尔型 (1种)**：用于存逻辑真假。**boolean** (只有 true 和 false)。

### 整型/**浮点型/字符型/布尔型**

#### 2. 重点补充与避坑指南

**A. 后缀规则（Suffix Rule）**

- **Long 型**：因为系统默认整数是 int，如果数字太大超过 int 范围，必须在末尾加 L 或 l。

- *例*：long num = 88888888888L;

- **Float 型**：因为系统默认小数是 double，如果想存入 float，必须在末尾加 F 或 f，否则报错。

- *例*：float price = 12.5f; （不加 f 会报错，提示不能把 double 硬塞给 float）。

**B. 字符型 char 的秘密**

- **Unicode**：Java 的 char 占用 **2 个字节**（16位），所以它**可以存储一个汉字**。

- *例*：char c = '中'; 是合法的。

- **本质是数字**：char 在底层是以数字存储的（ASCII 码或 Unicode 码），可以参与数学运算。

- *例*：'A' + 1 的结果是 66（因为 'A' 的 ASCII 码是 65）。

**C. 类型转换（Type Casting）**

- **自动类型转换（隐式）**：

- **小 ****→→**** 大**：安全，系统自动搞定。

- *顺序*：byte →→ short →→ int →→ long →→ float →→ double。

- **强制类型转换（显式）**：

- **大 ****→→**** 小**：危险，可能丢失精度，必须手动写括号。

- *格式*：int a = (int) 3.14; （结果 a 变成了 3，小数丢了）。

## 领会：

### 不同数据类型常量值的表示；

- **整型常量**：

- **默认**：直接写数字是 int（如 100）。

- **长整型**：数字后必须加 **L** 或 l（如 88888888888L）。

- *进制陷阱*：

- 以 **0** 开头是**八进制**（如 012 不是 12 而是 10）。

- 以 **0x** 开头是**十六进制**（如 0x1A）。

### 不同数据类型变量定义与赋值。

- **基本格式**：**数据类型 变量名 = 值;**

- **拆分格式**（先声明，后赋值）

```java
int age;    // 声明
age = 18;   // 赋值
```

- **同时定义多个**（必须同类型）：int x = 10, y = 20;

- **强制原则**：**局部变量使用前必须初始化（赋值）**，否则编译报错。

## 应用：

### 在程序中正确定义变量及赋值。


# 4. 数据输入与输出

## 识记：

### System.out类、Scanner类作用。

- **System.out**

- **作用**：**标准输出流**。

- **目的**：负责向**控制台**（屏幕）打印数据。

- **Scanner**

- **作用**：**文本扫描器**。

- **目的**：负责从**控制台**（键盘）获取用户输入的数据。

- **位置**：位于 **java.util** 包中（使用前必须导包）。

## 领会：

### System.out类、Scanner类使用方法。

**A. System.out 的核心方法**

- **System.out.print(内容)**：输出内容，**不换行**（光标停在后面）。

- **System.out.println(内容)**：输出内容，并**自动换行**（光标跳到下一行）。

- *技巧*：ln 代表 line（行）。

- **拼接符号**：使用 **+** 连接字符串和变量（如 "年龄：" + age）。

**B. Scanner 的使用“三部曲” (必背)**

- **导包**（在文件最开头）：import java.util.Scanner;

- **创建对象**（在 main 方法里）：Scanner sc = new Scanner(System.in);

- *考点*：**System.in** 代表标准输入（键盘），不能漏。

- **接收数据**（根据类型选方法）：

- 读整数：**int n = sc.nextInt();**

- 读小数：**double d = sc.nextDouble();**

- 读字符串：**String s = sc.next();遇到空格会截止 String s = sc.nextline();这个遇到空格不会截止；**

## 应用：

### 使用System.out类输出结果；**使用Scanner类及对应输入方法从控制台获取输入值。**

```java
// 1. 导包 (填空题常考)
import java.util.Scanner;

public class TestInput {
    public static void main(String[] args) {
        // 2. 创建 Scanner 对象 (注意 System.in)
        Scanner sc = new Scanner(System.in);

        // 使用 System.out 提示用户
        System.out.print("请输入你的年龄："); // print 不换行，输入体验更好

        // 3. 获取输入值 (注意数据类型匹配)
        int age = sc.nextInt(); 

        // 4. 使用 System.out 输出结果 (拼接字符串)
        System.out.println("明年你就 " + (age + 1) + " 岁了！");
    }
}
```

# 5. 运算符与表达式

## 识记：

### 各类运算符表示：算术运算符（+、-、*、/、%、--、++）；

- **基本**：+、-、*、/、%（取模/求余）。

- **自增自减**：++（加1）、--（减1）。

### 赋值运算符（+=、-=、*=、\=、%=）；

- **基本**：=（注意：这是赋值，不是相等）。

- **复合**：+=、-=、*=、/=、%=。

- *含义*：a += b 等价于 a = a + b。

### 关系运算符（>、<、>=、<=、==、!=）;

- >、<、>=、<=

- **相等判断**：**==** （两个等号）。

- **不等判断**：**!=**。

### 逻辑运算符（&&、||、|、&、|）；

- **与**：&& (短路与)、& (逻辑与)。

- **或**：|| (短路或)、| (逻辑或)。

- **非**：! (取反)。

### 条件运算符（条件？=表达式1：表达式2）

- **格式**：**条件 ? 结果1 : 结果2**

## 领会：

### 不同运算符作用；

#### 1. 算术运算的“三大坑”

- **除法 (****/****)**：整数除以整数，结果还是整数（**去尾法**）。*例*：5 / 2 结果是 **2**，不是 2.5。

- **取模 (****%****)**：结果的符号取决于**被除数**（前面的数）。*例*：-10 % 3 = -1；10 % -3 = 1。常用于判断奇偶数（n % 2 == 0）。

- **自增 (****++****)**：**前置 (****++i****)**：**先 +1，再使用**（先变后用）。**后置 (****i++****)**：**先使用，再 +1**（先用后变）。

#### 2. 逻辑运算的“短路效应” (必考)

- **&&**** (短路与)**：如果左边为 false，右边**不执行**，直接返回 false。

- **||**** (短路或)**：如果左边为 true，右边**不执行**，直接返回 true。*考点*：如果右边有 i++，发生了短路，i 的值就不会变。

#### 3. 赋值运算的“隐式转换”

- short s = 1;s = s + 1; ❌ **报错**（因为 1 是 int，结果变成 int，装不进 short）。s += 1; ✅ **通过**（系统自动帮你强转了）。

### 运算符优先级；

- **括号最牛**：()

- **单目运算**：++, --, !

- **算术运算**：* / % 高于 + -

- **关系运算**：>, <, ==

- **逻辑运算**：&& 高于 ||

- **赋值最弱**：=, +=

# 6. 数据类型转换

## 识记：

### 数据类型转换概念 

- **概念**：将一种数据类型的值转换为另一种数据类型。

- **铁律**：**boolean**** 类型不参与任何转换**，它自成一派。

## 领会：

### 两种类型转换方式；

**A. 自动类型转换（隐式转换）**

- **方向**：**小 ****→→**** 大** （范围小的转为范围大的）。

- **特点**：安全，系统**自动完成**，不需要写代码。

- **比喻**：把小杯水倒入大桶中，不会洒出来。

- **转换链（背诵）**：byte/short/char →→ **int** →→ long →→ float →→ double

**B. 强制类型转换（显式转换）**

- **方向**：**大 ****→→**** 小** （范围大的转为范围小的）。

- **特点**：有风险（可能**丢失精度**或**数据溢出**），必须**手动写代码**。

- **比喻**：把大桶水硬倒进小杯子，水会溢出（精度丢失）。

### 表达式中不同数据类型自动转换规则。

在进行算术运算（

- **底层提升**：byte、short、char 只要参与运算，**一律先自动变成 ****int** 再算。

- *考题*：byte a = 1; byte b = 2; byte c = a + b; →→ **报错！** 因为 a+b 结果是 int，装不进 byte。

- **最高级看齐**：整个表达式的结果类型，取决于参与运算的**最高级**类型。

- 如果有 double，结果就是 double。

- 如果有 float，结果就是 float。

- 如果有 long，结果就是 long。

## 应用：

### 在程序中实现数据类型强制转换。

```java
public class TypeCastTest {
    public static void main(String[] args) {
        // 场景 1：浮点转整型（去尾法）
        double d = 3.99;
        int i = (int) d; 
        System.out.println(i); // 输出 3（不是四舍五入，是直接砍掉小数）

        // 场景 2：大整数转小整型（溢出风险）
        int n = 129;
        byte b = (byte) n; 
        System.out.println(b); // 输出 -127（发生溢出，正数变负数，了解即可）

        // 场景 3：解决 byte 运算报错问题
        byte x = 10;
        byte y = 20;
        // byte z = x + y; // 报错
        byte z = (byte)(x + y); // 正确：先把算出来的 int 结果强转回 byte
    }
}
```
