## 01. 《寡人之于国也》（孟子）

| 词语 | 解释 |
| :--- | :--- |
| **寡人** | 古代国君对自己的谦称。 |

| **河内** | 魏国黄河以北地区。 |
| **凶** | 谷物收成不好，荒年。 |
| **河东** | 魏国黄河以东地区。 |
| **加** | 副词，更。 |
| **填然** | 拟声词，形容鼓声。 |
| **鼓之** | 动词，敲鼓。之，助词。 |
| **兵刃既接** | 武器已经接触，指交锋。 |
| **曳兵** | 拖着武器。 |
| **走** | 逃跑。 |
| **直** | 副词，仅，只。 |
| **是** | 代词，这。 |
| **胜**（shēng） | 尽。 |
| **数（cù）罟** | 细密的渔网。 |
| **洿池** | 低洼的池塘。 |
| **以时** | 按照季节。 |
| **养生丧死** | 养活着的人，埋葬死的人。 |
| **无憾** | 没有遗憾。 |
| **王道** | 儒家提倡的以仁政统治天下的政治主张。 |
| **树之以桑** | 种植桑树。 |
| **衣（yì）帛** | 穿丝织品（名作动）。 |
| **畜（xù）** | 畜养。 |
| **谨** | 认真办好。 |
| **庠序** | 古代学校。 |
| **申** | 反复陈述，反复教导。 |
| **孝悌** | 敬爱父母，敬爱兄长。 |
| **颁白** | 通“斑白”，头发花白。 |
| **负戴** | 背负头顶东西。 |
| **黎民** | 百姓。 |
| **王（wàng）** | 称王天下（名作动）。 |
| **检** | 约束，制止。 |
| **饿莩** | 饿死的人。 |
| **发** | 指开仓赈济百姓。 |
| **岁** | 年成。 |
| **罪岁** | 归罪于年成。 |
| **斯** | 连词，那么，这样。 |
| **至** | 归顺，投奔。 |

| **谨严** | 谨慎严密。 |
