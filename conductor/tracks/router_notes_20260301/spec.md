# Specification: 制作交换与路由技术的笔记 (Track: router_notes_20260301)

## Overview
在 `C4=归档资料/4.1=学习类/4.1.2=计算机专业/E交换与路由技术/` 目录下，根据自考大纲制作一份详尽的《路由器配置与路由协议》学习笔记。该笔记将作为自考备考的核心资料。

## Functional Requirements
1. **结构化笔记内容：**
   - 笔记必须严格按照大纲中的 8 个核心知识点组织：
     1. 模拟器中路由器的配置
     2. 路由器的基本配置
     3. DHCP 动态获取地址
     4. 单臂路由实现部门间互访
     5. 静态路由配置
     6. 默认路由及浮动路由配置
     7. RIPv2 动态路由协议
     8. OSPF 动态路由协议
   - 每个知识点需包含：**识记**、**领会**、**应用** 三个子章节。

2. **多维内容支持：**
   - **MarkMind 脑图：** 在笔记中集成或配套生成一个脑图文件，展示本章的逻辑架构。
   - **CLI 命令块：** 针对“应用”部分，提供具体的路由器配置 CLI 命令（如 Cisco 或 H3C 风格）。
   - **图片占位符：** 在需要模拟器拓扑或运行结果的地方添加 `[截图占位: 描述]`。

## Non-Functional Requirements
- **语法规范：** 符合 Obsidian 链接规范及标准 Markdown 语法。
