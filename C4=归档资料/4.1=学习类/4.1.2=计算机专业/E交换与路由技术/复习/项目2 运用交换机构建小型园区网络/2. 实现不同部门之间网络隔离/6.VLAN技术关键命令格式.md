掌握 VLAN 的关键配置命令，是你把理论转化为实际操作的最后一关。在四月份的考场上，无论是面对 eNSP 仿真大题，还是纸质卷子上的填空与排错，这套命令代码都是绝对的“送分题”和网络构建的基石。

以下为你整理了 VRP 系统中最核心、最高频的 VLAN 配置命令字典。建议你在 eNSP 里一边敲击一边记忆：

### 1. VLAN 的创建与删除（系统视图下）

在给端口分配 VLAN 之前，**必须先在系统全局创建这个 VLAN**，否则系统会报错。

- **创建单个 VLAN：**
    
    Plaintext
    
    ```
    [Huawei] vlan 10               // 创建 VLAN 10 并自动进入 VLAN 10 视图
    [Huawei-vlan10] description HR // (可选) 给这个 VLAN 写个备注，比如人力资源部
    [Huawei-vlan10] quit           // 退回系统视图
    ```
    
- **批量创建 VLAN（考试提速神技）：**
    
    Plaintext
    
    ```
    [Huawei] vlan batch 10 20 30   // 一次性同时创建 VLAN 10, 20, 30
    [Huawei] vlan batch 100 to 200 // 一次性创建 VLAN 100 到 VLAN 200 之间的所有连续 VLAN
    ```
    
- **删除 VLAN：**
    
    Plaintext
    
    ```
    [Huawei] undo vlan 10          // 删除 VLAN 10
    ```
    

### 2. Access 端口配置（连接 PC 终端）

进入连接电脑的具体物理接口，将其设为接入模式，并划分到指定的 VLAN 中。

Plaintext

```
[Huawei] interface GigabitEthernet 0/0/1
[Huawei-GigabitEthernet0/0/1] port link-type access  // 步骤 1：将端口类型改为 Access
[Huawei-GigabitEthernet0/0/1] port default vlan 10   // 步骤 2：将该端口划入 VLAN 10（即修改 PVID 为 10）
[Huawei-GigabitEthernet0/0/1] quit
```

### 3. Trunk 端口配置（连接其他交换机或路由器）

配置主干道链路，允许带有特定标签的数据帧通过。

Plaintext

```
[Huawei] interface GigabitEthernet 0/0/24
[Huawei-GigabitEthernet0/0/24] port link-type trunk               // 步骤 1：将端口类型改为 Trunk
[Huawei-GigabitEthernet0/0/24] port trunk allow-pass vlan 10 20   // 步骤 2：放行 VLAN 10 和 VLAN 20 的数据通过
// (如果想允许所有 VLAN 通过，可以使用 port trunk allow-pass vlan all，但真实工程中为了安全通常不这么做)
[Huawei-GigabitEthernet0/0/24] quit
```

> **⚠️ 极易丢分点：** 很多考生配完 `link-type trunk` 后就以为万事大吉了，**忘记敲 `allow-pass vlan`**。在华为设备上，Trunk 端口默认只允许 VLAN 1 通过，如果不手动放行其他 VLAN，跨交换机的通信必断无疑！

### 4. 终极排错与验证命令（任意视图下）

配置完成后，千万不要去盲目 ping。养成查表的习惯，能瞬间定位故障点。

- **查看全局 VLAN 简表（最常用）：**
    
    Plaintext
    
    ```
    [Huawei] display vlan
    ```
    
    _作用：列出当前设备上创建了哪些 VLAN，以及每个 VLAN 里面包含哪些物理接口。如果发现某个接口没在对应的 VLAN 里，立刻就能知道是端口划分配错了。_
    
- **查看所有接口的 VLAN 状态：**
    
    Plaintext
    
    ```
    [Huawei] display port vlan
    ```
    
    _作用：以列表形式清晰展示每个接口当前的链路类型（Link Type 是 Access 还是 Trunk）、PVID 是多少、以及允许哪些 VLAN 通过（Trunk VLAN List）。这是排查 Trunk 漏配的最佳命令。_
