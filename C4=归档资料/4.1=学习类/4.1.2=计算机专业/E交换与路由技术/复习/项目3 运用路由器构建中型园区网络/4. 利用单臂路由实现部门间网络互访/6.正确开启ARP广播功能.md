安排！这是单臂路由中极其重要、极易遗漏的一步。

在华为 VRP 系统中，**物理接口默认是开启 ARP 广播的，只有“逻辑子接口”需要手动开启**。

为了你的笔记和上机提速，以下是纯简写版的配置和验证指令：

---

# 核心考点笔记：正确开启 ARP 广播 (纯简写版)

## 一、核心配置指令

必须进入对应的**子接口视图**下敲击：

Plaintext

```
[R1] int g0/0/0.10             // 1. 进入你需要开启的子接口
[R1-GigabitEthernet0/0/0.10] arp b e   // 2. 核心简写：开启 ARP 广播 (arp broadcast enable)
```

## 二、黄金自测验证 (`dis th`)

在子接口下敲完后，千万别急着退出来，顺手敲一个 `dis th` (display this) 检查一下当前接口的配置：

Plaintext

```
[R1-GigabitEthernet0/0/0.10] dis th
```

_👉 只要在输出结果中看到 `arp broadcast enable` 这一行，就说明成功开启，底下 PC 找网关绝对没问题了。_
