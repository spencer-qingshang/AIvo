# Java核心知识点速查手册 (1-8章 详细版)

本手册根据您的要求，在关键字的基础上，增加了必须掌握的核心类及其最重要的方法，以便于您进行高效的冲刺复习。

---

## 第一章：初识Java

| 关键字/命令 | 核心作用/解释                                  |
| :------------ | :--------------------------------------------- |
| `class`       | 定义一个类，是Java程序的基本单元。             |
| `public`      | 访问修饰符，表示公开，任何地方都可访问。       |
| `static`      | 静态修饰符，表示成员属于类，而非某个具体对象。 |
| `void`        | 返回值类型，表示方法不返回任何值。               |
| `main`        | 特殊方法名，是Java程序的唯一入口点。           |
| `javac`       | **命令**：用于**编译** `.java` 源代码文件。      |
| `java`        | **命令**：用于**运行**编译后的 `.class` 字节码文件。 |

## 第二章：Java编程基础

#### 1. 核心关键字

| 关键字  | 核心作用/解释                                                |
| :------ | :----------------------------------------------------------- |
| `final` | 修饰符，修饰的变量变为常量（只能赋值一次），修饰的类不能被继承，修饰的方法不能被重写。 |
| `import`| 导入其他包中的类，以便在当前文件中使用。                 |
| `new`   | 创建对象（实例）的关键字。                                   |

#### 2. 基本数据类型

| 关键字    | 占用空间 | 范围/特点                                   |
| :-------- | :------- | :------------------------------------------ |
| `byte`    | 1字节    | -128 到 127                                 |
| `short`   | 2字节    | -32768 到 32767                             |
| `int`     | 4字节    | 约±21亿，是整型的默认类型。                 |
| `long`    | 8字节    | 极大，数值后需加`L`或`l`。                  |
| `float`   | 4字节    | 单精度浮点型，数值后需加`F`或`f`。          |
| `double`  | 8字节    | 双精度浮点型，是浮点型的默认类型。          |
| `char`    | 2字节    | 16位Unicode字符型，可存单个汉字。           |
| `boolean` | -        | 逻辑类型，只有`true`和`false`两个值。       |

#### 3. 输入/输出核心类与方法

| 类/对象      | 方法                  | 核心作用/解释                              |
| :----------- | :-------------------- | :----------------------------------------- |
| `System.out` | `print()` / `println()` | 向控制台输出信息，后者会自动换行。         |
| `Scanner`    | `new Scanner(System.in)` | 创建一个扫描器对象以读取键盘输入。         |
|              | `nextInt()`           | 读取并返回一个整数。                       |
|              | `nextDouble()`        | 读取并返回一个双精度浮点数。               |
|              | `next()`              | 读取一个字符串（遇到空格、Tab或回车结束）。 |
|              | `nextLine()`          | 读取一整行字符串（直到遇到回车结束）。     |

## 第三章：流程控制与方法

| 关键字     | 核心作用/解释                                                |
| :--------- | :----------------------------------------------------------- |
| `if` / `else` | 分支结构关键字，用于条件判断。                             |
| `switch` / `case` / `default` | 分支结构关键字，用于固定值匹配。                           |
| `break`    | 在`switch`中用于跳出结构；在循环中用于**彻底终止**整个循环。 |
| `for` / `while` / `do-while` | 循环结构关键字，用于重复执行代码。                         |
| `continue` | **跳过**本次循环中余下的代码，直接进入下一次循环。         |
| `return`   | 结束方法的执行，并可返回一个值。                           |

## 第四章：数组

| 语法/类      | 方法/属性          | 核心作用/解释                                                |
| :----------- | :----------------- | :----------------------------------------------------------- |
| `[]`         | `数组名[索引]`     | 用于定义数组类型，以及访问指定索引的数组元素。             |
| (数组)     | `.length`          | **属性**：获取数组的长度（元素个数）。注意没有括号。       |
| `Arrays`     | `toString(数组)`   | **静态方法**：将数组内容转换为字符串格式（如 `[1, 2, 3]`）。 |
|              | `sort(数组)`       | **静态方法**：对数组进行升序排序。                         |
|              | `binarySearch(数组, 值)` | **静态方法**：二分查找（**前提：数组必须已排序**）。       |

## 第五/六章：面向对象 (继承、多态、抽象类、接口)

#### 1. 核心关键字

| 关键字      | 核心作用/解释                                                |
| :---------- | :----------------------------------------------------------- |
| `private`   | 访问修饰符，表示私有，只能在本类中访问，是封装的核心。     |
| `protected` | 访问修饰符，表示受保护，本包及外包子类可访问。             |
| `this`      | 代表**当前对象**的引用，用于区分同名变量或调用本类构造器。 |
| `extends`   | 继承关键字，一个类继承另一个类。                             |
| `super`     | 代表**父类**的引用，用于调用父类的构造器或成员。           |
| `@Override` | **注解**：表示该方法是重写父类的方法。                     |
| `instanceof`| **运算符**：判断一个对象是否是某个类（或其子类）的实例。   |
| `abstract`  | **抽象修饰符**：修饰类（抽象类）或方法（抽象方法）。         |
| `interface` | 定义一个**接口**。                                           |
| `implements`| **实现**关键字，一个类实现一个或多个接口。                 |

#### 2. 异常处理

| 关键字    | 核心作用/解释                                                |
| :-------- | :----------------------------------------------------------- |
| `try`     | 包裹可能发生异常的代码块。                                   |
| `catch`   | 捕获并处理在`try`块中发生的异常。                            |
| `finally` | 无论是否发生异常，总会执行的代码块，常用于释放资源。       |
| `throw`   | 在**方法体内部**手动创建并**抛出**一个异常对象。             |
| `throws`  | 在**方法声明处****声明**该方法可能抛出的异常类型。           |

## 第七章：常用核心类

| 类名             | 方法/属性                | 核心作用/解释                                                |
| :--------------- | :----------------------- | :----------------------------------------------------------- |
| `Object`         | `toString()`             | 返回对象的字符串表示，常被重写。                           |
|                  | `equals(Object obj)`     | 比较两个对象是否相等，常被重写以比较内容。                 |
| `String`         | `.length()`              | 获取字符串长度。                                             |
|                  | `.equals(Object obj)`    | 比较字符串**内容**是否相等。                               |
|                  | `.charAt(int index)`     | 获取指定索引位置的字符。                                     |
|                  | `.substring(int begin, int end)` | 截取子字符串（含头不含尾）。                               |
|                  | `.split(String regex)`   | 根据指定的分隔符分割字符串，返回一个字符串数组。           |
| `StringBuffer`   | `.append(值)`            | 在字符串末尾追加内容。                                       |
|                  | `.insert(int offset, 值)` | 在指定位置插入内容。                                         |
|                  | `.reverse()`             | 反转字符串。                                                 |
| `Integer`等包装类 | `Integer.parseInt(String s)` | **静态方法**：将字符串解析为`int`类型。                     |
| `Math`           | `Math.random()`          | **静态方法**：生成 `[0.0, 1.0)` 范围的`double`随机数。     |
| `Random`         | `nextInt(int n)`         | 生成 `[0, n)` 范围的`int`随机数。                          |
| `Date`           | `new Date()`             | 创建一个表示当前日期和时间的对象。                         |
| `Calendar`       | `getInstance()`          | **静态方法**：获取一个表示当前日期和时间的日历对象。       |
|                  | `get(int field)`         | 获取日历中的特定字段（如`Calendar.YEAR`）。                |
|                  | `add(int field, int amount)` | 对指定字段增减时间。                                         |
| `SimpleDateFormat` | `format(Date date)`      | 将`Date`对象**格式化**为指定模式的字符串。                 |
|                  | `parse(String source)`   | 将指定模式的字符串**解析**为`Date`对象。                   |

## 第八章：集合框架

| 接口/类/方法     | 核心作用/解释                                                |
| :--------------- | :----------------------------------------------------------- |
| `List`           | `Collection`的子接口，特点：**有序、可重复**。                 |
| `Set`            | `Collection`的子接口，特点：**无序、不可重复**。               |
| `Map`            | 存储**键值对**的顶层接口，特点：**键唯一、值可重复**。       |
| `ArrayList`      | `List`接口的常用实现类，底层是数组，查询快。                 |
| `HashSet`        | `Set`接口的常用实现类，底层是哈希表，无序。                  |
| `HashMap`        | `Map`接口的常用实现类，底层是哈希表，无序。                  |
| `add(E e)`       | `Collection`方法：向集合中添加一个元素。                     |
| `get(int index)` | `List`方法：获取指定索引的元素。                             |
| `remove(Object o)`/`remove(int index)` | `Collection`/`List`方法：删除元素。                          |
| `size()`         | `Collection`/`Map`方法：获取集合或Map中元素的数量。          |
| `put(K key, V value)` | `Map`方法：添加或更新一个键值对。                          |
| `get(Object key)`| `Map`方法：根据键获取值。                                    |
| `keySet()`       | `Map`方法：获取所有**键**的`Set`集合，用于遍历。             |
| `entrySet()`     | `Map`方法：获取所有**键值对**的`Set`集合，用于高效遍历。   |
| `Iterator`       | **迭代器**接口，是遍历集合的标准方式。                       |
| `iterator()`     | `Collection`方法：返回一个`Iterator`对象。                   |
| `hasNext()` / `next()` | `Iterator`方法：判断是否有下一个元素 / 获取下一个元素。      |
| `Collections`    | 操作集合的**工具类**。                                     |
| `.sort(List list)` | `Collections`的静态方法，对`List`进行排序。                  |
