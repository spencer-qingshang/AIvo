# 1. 三种程序结构

## 识记：

### 三种程序结构概念：顺序结构、分支结构、循环结构。

- **顺序结构 (Sequential Structure)**

- **概念**：程序**从上到下**、**一行一行**地执行代码。

- **特点**：没有任何跳转或判断，是程序执行的**默认**流程。

- **分支结构 (Branching / Selection Structure)**

- **概念**：也叫**选择结构**。根据**条件**（True/False）判断，选择执行不同的代码块。

- **关键词**：if, else, switch。

- **循环结构 (Loop Structure)**

- **概念**：在满足特定条件下，**重复执行**某段代码。

- **关键词**：for, while, do-while。

## 领会：

### 三种程序结构作用及应用场景。

**A. 顺序结构**

- **作用**：完成最基本的、按部就班的操作。

- **场景**：

- 计算圆的面积（输入半径 →→ 计算 →→ 输出）。

- 交换两个变量的值（中间瓶换水）。

- **比喻**：**流水线**。一条道走到黑，不回头，不拐弯。

**B. 分支结构**

- **作用**：让程序具有**“逻辑判断”**能力，处理不同的情况。

- **场景**：

- **登录判断**：密码对 →→ 进首页；密码错 →→ 提示错误。

- **成绩分级**：分数 > 60 及格，否则不及格。

- **判断奇偶数**：num % 2 == 0。

- **比喻**：**十字路口**（或铁路道岔）。红灯停，绿灯行；向左走还是向右走。

**C. 循环结构**

- **作用**：**减少重复代码**，提高效率，处理大量数据。

- **场景**：

- **累加求和**：计算 1 到 100 的和。

- **遍历数组**：把班里 50 个人的名字都打印出来。

- **持续等待**：一直提示用户输入，直到输入正确为止。

- **比喻**：**操场跑圈**。只要体力够（条件满足），就一直跑；累了（条件不满足）就停下。

# 2. 分支结构

## 识记：

### 分支语法结构，包括if；

- **单分支**：if (条件) { 代码 }*逻辑*：满足条件就做，不满足就跳过。

### if-else；

- **双分支**：if (条件) { 代码A } else { 代码B }*逻辑*：二选一（非黑即白）。

### if-else多分支；

```java
if (条件1) { 代码A } 
else if (条件2) { 代码B } 
else { 代码C } // 兜底
```

### if嵌套；

if

### switch多分支。

```java
switch (变量) {
    case 值1:
        代码A;
        break; // 必考点：刹车
    case 值2:
        代码B;
        break;
    default:   // 必考点：兜底（相当于 else）
        代码C;
        break;
}
```

## 领会：

### 分支语法使用方法；

### 不同分支语法差异。

**A. if vs switch 的区别（必考选择题）**

- **if 语句**：

- 擅长处理**区间范围**（例如：score > 60 && score < 80）。

- 擅长处理**布尔值**（True/False）。

- **最通用**，switch 能做的 if 都能做。

- **switch 语句**：

- 擅长处理**固定值/离散值**的匹配（例如：星期几、菜单选项 A/B/C）。

- 结构更清晰，效率略高。

**B. Switch 的“三大雷区”（改错题高频）**

- **穿透现象**：如果漏写 **break**，程序会**继续向下执行**下一个 case 的代码，直到遇到 break 或大括号结束。（利用这个特性可以实现“多个 case 执行同一段代码”）。

- **支持类型（死记）**：

- ✅ 支持：byte, short, int, char, **String** (JDK7以后), enum。

- ❌ **不支持**：long, float, double, boolean。

- **default 位置**：可以在任意位置，但习惯放在最后。

## 应用：

### 能够针对不同需求采用不同分支语法实现分支结构程序代码编写。

```java
//场景 1：判断奇偶（双分支 if-else）
if (num % 2 == 0) {
    System.out.println("偶数");
} else {
    System.out.println("奇数");
}

//场景 2：成绩分级（多分支 if-else if）
//注意：范围大的写上面还是下面？要看逻辑，通常严苛的条件先判断。
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 60) {
    System.out.println("及格"); // 隐含了 < 90
} else {
    System.out.println("不及格");
}
//场景 3：根据数字打印星期（Switch）
int week = 3;
switch (week) {
    case 1: System.out.println("周一"); break;
    case 2: System.out.println("周二"); break;
    case 3: System.out.println("周三"); break; // 输出这个，然后跳出
    default: System.out.println("输入错误");
}
```

# 3. 循环结构

## 识记：

### 几种循环语法结构：while；do…while；for；continues；break。

**A. 三大循环语法**

- **for 循环**（最常用）

- **格式**：for (初始化; 条件判断; 步进迭代) { 循环体 }

- *执行顺序（必考）*：

- ① 初始化 (只跑一次) →→ ② 判断条件 →→ ③ (若真)执行循环体 →→ ④ 步进迭代 →→ ② 判断条件...

- **while 循环**

- **格式**：while (条件) { 循环体 }

- *逻辑*：先判断，后执行。

- **do...while 循环**

- **格式**：do { 循环体 } while (条件);

- *逻辑*：先斩后奏（先执行一次，再判断）。

- *考点*：**最后的分号 ****;**** 绝对不能漏**。

**B. 跳转语句**

- **break**：**终止/跳出**。彻底结束整个循环，去执行循环后面的代码。

- **continue**：**跳过/继续**。跳过**本次**循环剩下的代码，直接进入下一次循环。

## 领会：

### 不同循环语句差异。

**A. 三种循环的区别（选择题高频）**

- **for vs while**：

- **for**：适合**循环次数固定**的情况（如：从1加到100）。

- **while**：适合**循环次数不确定**，只知道停止条件的情况（如：直到用户输入正确密码为止）。

- **while vs do...while**：

- **while**：**先判断**。如果条件一开始就不满足，**一次都不执行**（0次）。

- **do...while**：**后判断**。无论条件如何，**至少执行一次**（1次）。

**B. break vs continue 的比喻**

- **break (辞职)**：彻底不干了，离开公司。

- **continue (请假)**：今天剩下的活不干了，明天（下一次）继续来上班。

## 应用：

### 能够采用相关循环语法编写代码，解决特定现实问题。

```java
//场景 1：累加求和 (固定次数 →→ for)
//任务：计算 1 到 100 的和。
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum += i; // 累加
}
System.out.println(sum);

//场景 2：密码验证 (次数未知 →→ while)
//任务：一直提示输入，直到输入 "123456"。
// 假设 sc 是 Scanner 对象
String pwd = "";
while (!"123456".equals(pwd)) { // 只要密码不对，就一直循环
    pwd = sc.next();
}

//场景 3：筛选输出 (continue)
//任务：输出 1-10 之间的所有偶数。
for (int i = 1; i <= 10; i++) {
    if (i % 2 != 0) {
        continue; // 如果是奇数，跳过本次打印，直接进入下一次
    }
    System.out.println(i);
} 
```

# 4. 方法

## 识记：

### 方法的概念、结构与作用。

- **概念**：方法是将具有独立功能的**代码块**组织成为一个整体，使其具有特殊功能的代码集。

- **作用**：

- **复用代码**：写一次，到处用（不用重复造轮子）。

- **简化程序**：让 main 方法更干净，逻辑更清晰。

```java
修饰符 返回值类型 方法名(参数列表) {
    方法体;
    return 返回值;
}
public static int add(int a, int b) { ... }
```

## 领会：

### 掌握定义方法的三个要素，返回值、方法名、参数，会定义方法

- **返回值类型 (Return Type)**：

- **有返回值**：方法执行完要给调用者一个结果（如 int, String）。**必须配合 ****return**** 关键字**。

- **无返回值**：只执行操作，不给结果（如打印一句话）。**必须写 ****void**。

- **方法名 (Method Name)**：

- 遵循**小驼峰**命名法（如 getSum），见名知意。

- **参数列表 (Parameter List)**：

- **形式参数**：定义时写的变量名（如 int a），相当于占位符。

- **实际参数**：调用时传入的具体值（如 10）。

## 应用：

### 能根据实际需求完成方法定义和调用；

**A. 方法的调用 (Calling)**

- **格式**：方法名(实参1, 实参2);

- **赋值调用**：int result = add(10, 20); （推荐，适合有返回值的方法）。

- **输出调用**：System.out.println(add(10, 20));。

- **直接调用**：printHello(); （适合 void 方法）。

**B. 递归 (Recursion)**

- **概念**：**方法自己调用自己**。

- **核心要点（必考）**：**递归出口**：必须有一个结束条件（死胡同的尽头），否则会变成**死循环**导致**栈内存溢出** (StackOverflowError)。**递归规律**：每一次调用都比上一次更接近出口。

- *经典案例*：阶乘（5! = 5 * 4!）。

### 了解递归实现。

```java
//场景 1：求两个数的最大值（有返回值）
// 定义方法
public static int getMax(int a, int b) {
    return (a > b) ? a : b; // 把结果扔回去
}

// 调用方法 (在 main 中)
int max = getMax(10, 20);
System.out.println(max);

//场景 2：递归求 5 的阶乘 (5 * 4 * 3 * 2 * 1)
public static int jieCheng(int n) {
    if (n == 1) {
        return 1; // 1. 出口：到 1 就停
    } else {
        return n * jieCheng(n - 1); // 2. 自己调自己
    }
}
```
