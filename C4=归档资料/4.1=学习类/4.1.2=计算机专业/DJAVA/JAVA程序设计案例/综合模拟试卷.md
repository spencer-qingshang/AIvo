## 第一部分：单项选择题

**1. (第一章)** 如果我们只需要运行一个已经编译好的Java程序，而不需要进行开发，那么我们最少只需要安装什么？
   A. JDK (Java Development Kit)
   B. JRE (Java Runtime Environment)
   C. JVM (Java Virtual Machine)
   D. Eclipse IDE

**2. (第五章)** 关于Java中的构造方法，以下说法**错误**的是？
   A. 构造方法名必须与类名完全相同。
   B. 构造方法没有返回值类型，连void都不能写。
   C. 如果一个类没有显式定义构造方法，Java会为其提供一个默认的无参构造方法。
   D. 构造方法可以被`static`关键字修饰。

**3. (第八章)** 下列关于`String`类的说法，正确的是？
   A. `String`对象的内容是可变的。
   B. `String s1 = "abc"; String s2 = "abc";` 这两条语句执行后，`s1 == s2` 的结果是 `false`。
   C. `String`类是 `final` 的，不能被继承。
   D. 可以使用 `new String("abc")` 创建一个 `StringBuffer` 对象。

**4. (第十一章)** 在Java中，创建多线程的方式不包括以下哪一种？
   A. 继承 `Thread` 类，并重写 `run()` 方法。
   B. 实现 `Runnable` 接口，并实现 `run()` 方法。
   C. 实现 `Callable` 接口，并实现 `call()` 方法。
   D. 继承 `Object` 类，并重写 `start()` 方法。

## 第四部分：程序设计题

**12. (第四章)** 编写一个Java方法，该方法接收一个整型数组 `int[] arr` 和一个整数 `int target`作为参数，返回 `target` 在数组中第一次出现的索引。如果数组中不存在该元素，则返回 -1。

   ```java
   // 请补充以下方法
   public int findIndex(int[] arr, int target) {
       // Your code here
   }
   ```

**13. (综合应用)** 请设计一个`Student`类，包含以下私有属性：
    *   `name` (String): 学生姓名
    *   `score` (double): 学生分数

    要求：
    1.  提供一个接收姓名和分数的构造方法。
    2.  为这两个属性提供公开的 `get` 和 `set` 方法。
    3.  重写 `toString()` 方法，使其返回类似 "姓名: 张三, 分数: 95.5" 的格式。
    4.  在`setScore`方法中添加逻辑判断：如果传入的分数小于0或大于100，则不进行赋值，并打印一条错误提示信息。
