## 四、 综合应用题 (每小题 10 分，共 20 分)

1.  **ACL 访问控制配置：**
    某园区网要求禁止市场部（192.168.10.0/24）的用户在上班时间访问财务服务器（172.16.1.100）的 Web 服务（端口 80），但允许其他访问。请补充 R1 上的高级 ACL 配置：
    ```shell
    [R1] acl 3005
    [R1-acl-adv-3005] rule deny tcp source ____________ (1) ____________ (2) destination 172.16.1.100 0 destination-port ______ (3) 80
    [R1-acl-adv-3005] quit
    [R1] interface GigabitEthernet 0/0/0  # 核心交换机下接市场部的接口
    [R1-GigabitEthernet0/0/0] traffic-filter ____________ (4) acl 3005
    ```

2.  **NAT 与 端口安全配置：**
    要求配置 R1 接口 G0/0/1 使用 Easy-IP 实现内网（ACL 2000 定义）上网；同时在接入交换机 SW1 的 G0/0/1 接口开启端口安全，限制最大 MAC 数为 1，动作设为 Restrict。
    ```shell
    # R1 NAT 配置
    [R1] acl 2000
    [R1-acl-basic-2000] rule permit source 192.168.1.0 0.0.0.255
    [R1] interface GigabitEthernet 0/0/1
    [R1-GigabitEthernet0/0/1] nat ____________ (5) 2000
    
    # SW1 端口安全配置
    [SW1] interface GigabitEthernet 0/0/1
    [SW1-GigabitEthernet0/0/1] port-security ____________ (6)
    [SW1-GigabitEthernet0/0/1] port-security max-mac-num ____________ (7)
    [SW1-GigabitEthernet0/0/1] port-security protect-action ____________ (8)
    ```

### 三、 简答题
1. 答：基本 ACL（2000-2999）仅能匹配源 IP；高级 ACL（3000-3999）可匹配源/目 IP、协议号、端口号等。
2. 答：Sticky MAC 可以自动学习 MAC 且重启后配置不丢失，兼顾了安全性与维护便利性。
3. 答：1.节省公网 IP；2.隐藏内部拓扑提高安全；3.解决私网地址重叠。
4. 答：PAP 为两次握手，明文发送密码；CHAP 为三次握手，密文交互，抗截获能力强。
5. 答：安全区域是防火墙衡量网络可信任程度的度量。名称：Local, Trust, DMZ, Untrust。
