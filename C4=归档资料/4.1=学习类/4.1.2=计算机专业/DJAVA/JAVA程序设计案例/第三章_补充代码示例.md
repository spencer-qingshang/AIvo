### 1. 分支结构 (`if-else` 和 `switch`) 示例

分支结构让程序可以根据不同的条件执行不同的路径。

#### `if-else if-else` 示例：成绩评级

`if` 结构非常适合处理**区间范围**的判断。

```java
public class IfElseExample {
    public static void main(String[] args) {
        int score = 85;
        char grade;

        if (score >= 90) {
            grade = 'A';
        } else if (score >= 80) {
            grade = 'B';
        } else if (score >= 60) {
            grade = 'C';
        } else {
            grade = 'D';
        }
        
        System.out.println("分数 " + score + " 对应的等级是: " + grade); // 输出: B
    }
}
```

#### `switch` 示例：星期判断

`switch` 结构非常适合处理**固定值**的匹配。

```java
public class SwitchExample {
    public static void main(String[] args) {
        int dayOfWeek = 3;
        String dayName;

        switch (dayOfWeek) {
            case 1:
                dayName = "星期一";
                break; // break 用于跳出 switch，如果没有它会“穿透”到下一个case
            case 2:
                dayName = "星期二";
                break;
            case 3:
                dayName = "星期三";
                break;
            case 4:
                dayName = "星期四";
                break;
            case 5:
                dayName = "星期五";
                break;
            case 6:
            case 7: // "穿透"特性：case 6和7执行同样的代码
                dayName = "周末";
                break;
            default: // 如果所有case都不匹配，则执行default
                dayName = "无效的日期";
                break;
        }
        System.out.println("今天是: " + dayName); // 输出: 星期三
    }
}
```

### 2. 循环结构 (`for`, `while`, `do-while`) 示例

循环结构用于重复执行代码。

```java
public class LoopExample {
    public static void main(String[] args) {
        // --- for 循环：适合固定次数的循环 ---
        // 计算 1 到 10 的和
        int sum = 0;
        for (int i = 1; i <= 10; i++) {
            sum += i;
        }
        System.out.println("1到10的和是: " + sum); // 输出: 55

        // --- while 循环：适合次数不确定的循环 ---
        // 模拟一个计数器，直到大于3才停止
        int count = 0;
        while (count <= 3) {
            System.out.println("while循环，当前count: " + count); // 会输出 0, 1, 2, 3
            count++;
        }

        // --- do-while 循环：至少执行一次 ---
        // 即使条件一开始就不满足，循环体也至少执行一次
        int number = 100;
        do {
            System.out.println("do-while循环执行了一次。"); // 会输出一次
        } while (number < 10);
    }
}
```

### 3. 跳转语句 (`break` 和 `continue`) 示例

`break` 和 `continue` 用于在循环中提供更灵活的控制。

```java
public class JumpExample {
    public static void main(String[] args) {
        System.out.println("--- break 示例 ---");
        // 在1到10中寻找数字7，找到就停止
        for (int i = 1; i <= 10; i++) {
            if (i == 7) {
                System.out.println("找到了！就是 " + i);
                break; // 立即终止整个for循环
            }
            System.out.println("正在检查 " + i);
        }
        // 输出: 正在检查 1 ... 正在检查 6, 找到了！就是 7
        
        System.out.println("\n--- continue 示例 ---");
        // 打印1到10之间的所有奇数
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) { // 如果是偶数
                continue; // 跳过本次循环剩下的代码，直接进入下一次循环
            }
            System.out.print(i + " "); // 只会打印奇数
        }
        // 输出: 1 3 5 7 9 
    }
}
```

### 4. 方法 (Method) 与递归 (Recursion) 示例

方法是代码复用的基本单元，递归是一种特殊的调用方式。

```java
public class MethodAndRecursionExample {

    // 1. 定义一个普通方法：接收两个整数，返回它们的和
    public static int add(int a, int b) {
        return a + b;
    }
    
    // 2. 定义一个void方法：不返回任何值，只执行操作
    public static void printSeparator() {
        System.out.println("\n----------");
    }

    // 3. 定义一个递归方法：计算阶乘 (n!)
    // n! = n * (n-1) * ... * 1
    public static long factorial(int n) {
        // 递归出口：当n=1时，结束递归
        if (n == 1) {
            return 1;
        } else {
            // 递归规律：调用自身，但问题规模减小 (n-1)
            return n * factorial(n - 1);
        }
    }

    public static void main(String[] args) {
        // 调用普通方法
        int sum = add(10, 20);
        System.out.println("10 + 20 = " + sum); // 输出: 30

        printSeparator(); // 调用void方法
        
        // 调用递归方法
        long result = factorial(5); // 5! = 5 * 4 * 3 * 2 * 1
        System.out.println("5的阶乘是: " + result); // 输出: 120
    }
}
```
