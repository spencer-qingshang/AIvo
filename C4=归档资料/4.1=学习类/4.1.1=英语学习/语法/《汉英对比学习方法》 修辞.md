# 修辞的两个方面

比喻、夸张、排比属于修辞格。

修辞还有另一个方面，如何选择、运用词语和句式，以便使语言表达更准确、清除、恰当、有力和生动。

我们在平时说话、写作时，总汇希望表达的效果更好一些，而要做到这一点，就要注意措词，还要讲究表达方式。

# “文辞绚丽”与flowery language

英语喜欢平实的语言，汉语喜欢华丽的语言。

英语中华丽的语言，形容词修饰名词，副词修饰动词，形容词和副词，都可以用来强调语气、增强力度、烘托气氛，起渲染、夸饰作用。

# 强调性副词

中国学生喜欢用very，因为汉语经常使用“很”，但是外国人不喜欢用。

平时需要注意动词上如果强调意义足够强，那么就不需要使用这种强调副词。

# 强调性形容词

很多名词上已经包含了汉语里要表达的形容词的意思，所以不需要加形容词。

# 学会使用强势词

少用强调副词，寻找同义词。

# 汉字四字短语

欧美人士不习惯四字短语，不要直译。

# 重复

使用同义词或近义词时避免英语重复的办法之一。

# 比喻的使用

博喻就是用不同几个喻体从不同的角度反复设喻来说明同一个本体。

比喻这个东西属“情”的产物，和“理”关系不大；而中国人重“情”​，西方人重“理”​。重“情”者，总不愿将个人认为正确的东西主动讲给别人听，要别人接受，而是借用“比”的手法，拐个弯说，让别人去领悟。读者自己悟出来了，也就自然心甘情愿地接受了。西方人重“理”​，总是想以最严谨、最准确的逻辑形式和理性思考去说服对方，那自然不能过多地依靠比喻，因为比喻终归是不准确的，希望用比喻来把理说清说透绝不是“高招”​，所以中西方就有了这个差别。
