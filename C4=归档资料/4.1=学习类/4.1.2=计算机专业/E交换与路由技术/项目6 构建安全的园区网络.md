# 项目 6：构建安全的园区网络 (大纲同步版)

## 考点 1：交换机端口安全 (Port Security)

**1. 基本作用**：
限制交换机接口连接的电脑数量，防止非法设备接入。

**2. 核心参数**：
*   **默认数量**：开启端口安全后，默认限制只能连接 **1** 台电脑（MAC 地址数为 1）。
*   **Sticky MAC (粘性 MAC)**：让交换机自动记住连接的电脑。**优点**：保存配置后重启，交换机依然认识这台电脑，不需要重新学习。

**3. 惩罚动作 (如果违规了怎么办)**：
*   **Protect**：丢包，但不告警。
*   **Restrict (最常考)**：丢包，同时发送告警日志，并记录违规次数。
*   **Shutdown**：直接关闭接口（接口显示为 Error-down）。

**4. 常用命令**：
```shell
[SW-G0/0/1] port-security enable             # 开启功能
[SW-G0/0/1] port-security max-mac-num 1      # 限制数量为1
[SW-G0/0/1] port-security protect-action restrict # 设置动作为告警
```

## 考点 2：设备远程管理 (AAA 验证)

**1. 验证模式对比**：
*   **Password 模式**：只输密码。简单但不安全。
*   **AAA 模式 (最常考)**：输用户名 + 密码。支持权限分级（0-15级），安全性高。

**2. 管理协议对比**：
*   **Telnet**：明文传输，不安全。
*   **SSH (STelnet)**：加密传输，安全。

**3. 常用命令**：
```shell
[R1] aaa
[R1-aaa] local-user admin password cipher Huawei@123  # 建用户设密码
[R1-aaa] local-user admin privilege level 15         # 给最高权限15
[R1-aaa] local-user admin service-type telnet        # 允许Telnet
```

## 考点 3：ACL 访问控制列表

**1. 匹配规则**：
*   **从小到大，命中即止**：从 rule 5 开始看，只要匹配上了一条，后面就不看了。

**2. ACL 分类 (必考编号)**：
*   **基本 ACL (2000-2999)**：只能根据 **源 IP** 过滤流量。
*   **高级 ACL (3000-3999)**：可以根据 **源 IP、目的 IP、协议号、端口号** 过滤。

**3. 通配符掩码 (怎么写网段)**：
*   **0 表示匹配**（必须一模一样）；**1 表示忽略**。
*   例子：`192.168.1.0 0.0.0.255` 表示匹配 192.168.1.0 这个网段。

## 考点 4：NAT 网络地址转换

**1. 主要优点**：
*   **节省公网 IP**（最重要的作用）。
*   隐藏内部网络，更安全。

**2. 核心技术**：
*   **Easy-IP**：让全公司的人共用一个公网接口 IP 上网。
*   **NAT Server (静态 NAT)**：把内网服务器发布到公网，让外网人能访问。

**3. 常用命令 (Easy-IP)**：
```shell
[R1] acl 2000
[R1-acl-basic-2000] rule permit source 192.168.1.0 0.0.0.255 # 定义哪些人能上网
[R1] interface G0/0/1 (外网口)
[R1-G0/0/1] nat outbound 2000                                # 开启上网转换
```

## 考点 5：PPP 认证 (广域网安全)

**1. PAP 认证**：
*   **两次握手**，明文发送密码。安全性低。

**2. CHAP 认证 (最常考)**：
*   **三次握手**，密文发送。安全性高。

---

## 考点 6：防火墙基础

**1. 安全区域与优先级 (必考)**：
*   **Local (100)**：防火墙自己。
*   **Trust (85)**：内网。
*   **DMZ (50)**：放置服务器的区域。
*   **Untrust (5)**：外网（互联网）。

**2. 核心逻辑**：
*   华为防火墙 **默认拒绝所有流量**，必须手动写“安全策略”才能通行。
