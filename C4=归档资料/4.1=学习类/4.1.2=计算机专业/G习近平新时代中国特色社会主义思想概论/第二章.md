# 习近平新时代中国特色社会主义思想概论 - 第二章

## 📚 考核知识点与核心笔记

### 1. 中华民族近代以来最伟大的梦想

> **🎯 官方大纲要求**
> *   **识记**：
>     *   实现**中华民族伟大复兴**是近代以来最伟大的梦想。
>     *   **全面建成小康社会**。
>     *   打赢**脱贫攻坚战**。
>     *   **全面建成社会主义现代化强国**。
> *   **领会**：
>     *   中国共产党自成立起就肩负实现复兴的历史使命。
>     *   中国梦本质：**国家富强、民族振兴、人民幸福**。
>     *   如期全面建成小康社会（的意义）。
>     *   全面建成强国的**战略安排**。
> *   **应用**：
>     *   中国梦是国家的、民族的，也是每个人的。
>     *   中国梦是**和平、发展、合作、共赢**的梦。
>     *   全面建成社会主义现代化强国的**战略安排**。

#### (1) 中国梦的内涵与本质（领会/应用）
*   **本质**：国家富强、民族振兴、人民幸福。
*   **特性**：中国梦把国家、民族、个人的利益融为一体。它不仅是宏大叙事，也让每个人享有**出彩和成真**的机会。
*   **对外**：中国梦是**和平、发展、合作、共赢**的梦，主张与世界持久和平。

#### (2) 全面建成小康社会（识记/领会）
*   **里程碑**：2021年7月1日宣告全面建成。
*   **脱贫成就**：历史性解决了**绝对贫困**问题（近1亿人脱贫），创造了减贫奇迹。
*   **意义**：实现了“三步走”战略第二步，兑现了庄严承诺。

#### (3) 现代化强国战略安排（领会/应用）
**两步走战略**：
1.  **2020年到2035年**：**基本实现**社会主义现代化。
2.  **2035年到本世纪中叶**：**全面建成**富强民主文明和谐美丽的社会主义现代化强国。

### 2. 中国式现代化：唯一正确道路

> **🎯 官方大纲要求**
> *   **识记**：中国式现代化是党领导人民**长期探索和实践**的重大成果。
> *   **领会**：
>     *   “四个既...又...”：基于国情又借鉴经验，传承文化又融合文明。
>     *   **唯一正确道路**：中国式现代化走得通、行得稳。
>     *   中国式现代化的**中国特色**。
> *   **应用**：
>     *   中国式现代化的**本质要求**。
>     *   中国式现代化创造了**人类文明新形态**。

#### (1) 五个“中国特色”（领会）
1.  **人口规模巨大**（超过发达国家总和，难度空前）。
2.  **全体人民共同富裕**（坚持以人民为中心，防止两极分化）。
3.  **物质文明和精神文明相协调**（既要物质富足，也要精神富有）。
4.  **人与自然和谐共生**（绿水青山就是金山银山）。
5.  **走和平发展道路**（摒弃殖民掠夺老路）。

#### (2) 本质要求（应用·必背）
*   **核心领导**：坚持中国共产党领导。
*   **核心道路**：坚持中国特色社会主义。
*   **五大发展目标**：高质量发展、全过程人民民主、丰富精神世界、共同富裕、人与自然和谐共生。
*   **世界贡献**：推动构建人类命运共同体、创造人类文明新形态。

#### (3) 人类文明新形态（应用）
*   **重大超越**：是对西方式现代化理论和实践的重大超越。
*   **打破迷思**：打破了“现代化=西方化”的迷思。
*   **全新选择**：为发展中国家独立自主迈向现代化提供了全新选择。

### 3. 推进中国式现代化行稳致远

> **🎯 官方大纲要求**
> *   **识记**：**团结奋斗**是显著精神标识，是创造历史伟业的**必由之路**。
> *   **领会**：推进中国式现代化需要牢牢把握的**重大原则**。
> *   **应用**：推进中国式现代化需要正确处理的**重大关系**。

#### (1) 五条重大原则（领会·必背）
1.  坚持和加强**党的全面领导**（主心骨）。
2.  坚持**中国特色社会主义道路**（方向）。
3.  坚持**以人民为中心**的发展思想（价值）。
4.  坚持**深化改革开放**（动力）。
5.  坚持**发扬斗争精神**（保障）。

#### (2) 六个重大关系（应用）
1.  **顶层设计**与**实践探索**。
2.  **战略**与**策略**。
3.  **守正**与**创新**。
4.  **效率**与**公平**。
5.  **活力**与**秩序**。
6.  **自立自强**与**对外开放**。
