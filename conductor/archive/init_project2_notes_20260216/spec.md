# Specification: Init Project 2 Notes for Switching and Routing

## Overview
本项目旨在为“交换与路由技术”课程的“项目2 运用交换机构建小型园区网络”建立结构化的学习笔记。该笔记将作为 Obsidian 知识库中自考计算机专业核心资料的一部分。

## Target File
- `C4=归档资料/4.1=学习类/4.1.2=计算机专业/E交换与路由技术/项目2 运用交换机构建小型园区网络.md`

## Functional Requirements
1. **内容填充**：将原始资料中的文本按照 Markdown 结构化组织到目标文件中。
2. **结构组织**：
    - **一、学习目的与要求**：包含背景介绍和 9 项具体学习目标。
    - **二、考核知识点与考核要求**：详细列出 8 个知识点及其对应的“识记、领会、应用”要求。
    - **三、本章重点、难点**：概括核心理论与实践挑战。
    - **四、实战演练（预留）**：为后续添加实验配置命令、拓扑图和实验结果留出空间。
3. **排版要求**：使用清晰的标题层级（H1-H3）和列表（有序/无序），确保在 Obsidian 中易于阅读。

## Non-Functional Requirements
- **兼容性**：符合 Obsidian 的 Markdown 渲染规范。
- **一致性**：与现有“项目1”等笔记的风格保持一致。

## Acceptance Criteria
- [ ] 目标文件不再为空。
- [ ] 包含所有 4 个主要章节。
- [ ] 8 个考核知识点的细节完整无误。
- [ ] 文本中没有任何乱码或格式错误。

## Out of Scope
- 本阶段不包含具体的实验配置命令（将在实战演练手册中体现）。
- 不包含拓扑图的绘制。
