# 核心考点笔记：单臂路由满分配置模板 (纯简写版)

**实验场景设定：**

- **VLAN 10：** 网段 `192.168.10.0/24`，网关 `.254`。
    
- **VLAN 20：** 网段 `192.168.20.0/24`，网关 `.254`。
    
- **拓扑：** 交换机 G 0/0/1 连路由器 G 0/0/0。交换机 G 0/0/2 连 PC 1 (VLAN 10)，G 0/0/3 连 PC 2 (VLAN 20)。
    

---

## 第一步：交换机配置 (划分 VLAN + 放行 Trunk)

Plaintext

```
<Huawei> sys
[Huawei] sysn SW1

// 1. 批量创建 VLAN
[SW1] vlan bat 10 20

// 2. 配置连接 PC 的接口 (Access 模式)
[SW1] int g0/0/2
[SW1-GigabitEthernet0/0/2] p l a       // 简写: port link-type access
[SW1-GigabitEthernet0/0/2] p d v 10    // 简写: port default vlan 10
[SW1-GigabitEthernet0/0/2] q

[SW1] int g0/0/3
[SW1-GigabitEthernet0/0/3] p l a
[SW1-GigabitEthernet0/0/3] p d v 20
[SW1-GigabitEthernet0/0/3] q

// 3. 核心：配置连接路由器的单臂接口 (Trunk 模式)
[SW1] int g0/0/1
[SW1-GigabitEthernet0/0/1] p l t       // 简写: port link-type trunk
[SW1-GigabitEthernet0/0/1] p t a v 10 20 // 简写: port trunk allow-pass vlan 10 20
[SW1-GigabitEthernet0/0/1] q
```

---

## 第二步：路由器配置 (子接口 + 封装 + ARP 广播)

Plaintext

```
<Huawei> sys
[Huawei] sysn R1

// 1. 配置 VLAN 10 的子接口
[R1] int g0/0/0.10
[R1-GigabitEthernet0/0/0.10] dot1q term vid 10  // 核心: 识别并剥离 VLAN 10 标签
[R1-GigabitEthernet0/0/0.10] ip add 192.168.10.254 24 // 配置网关 IP
[R1-GigabitEthernet0/0/0.10] arp b e            // 灵魂简写: arp broadcast enable (开启ARP广播)
[R1-GigabitEthernet0/0/0.10] q

// 2. 配置 VLAN 20 的子接口
[R1] int g0/0/0.20
[R1-GigabitEthernet0/0/0.20] dot1q term vid 20
[R1-GigabitEthernet0/0/0.20] ip add 192.168.20.254 24
[R1-GigabitEthernet0/0/0.20] arp b e
[R1-GigabitEthernet0/0/0.20] q
```

---

## 第三步：必杀验证命令

敲完上述代码后，在路由器上敲这条命令验证：

- **`dis ip int br`** （全称：`display ip interface brief`）
    
    _👉 检查：看 `.10` 和 `.20` 两个子接口的状态是不是双 `Up`。_
    
- **`dis ip rou`** （全称：`display ip routing-table`）
    
    _👉 检查：看路由表里有没有自动生成 `192.168.10.0` 和 `192.168.20.0` 的直连（Direct）路由。_
