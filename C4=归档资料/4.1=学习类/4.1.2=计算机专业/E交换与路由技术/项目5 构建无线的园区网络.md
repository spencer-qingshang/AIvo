# 项目5 构建无线的园区网络 (深度备考版)

## 一、 核心概念对比 (必考识记)

### 1.1 Fat AP (胖AP) vs Fit AP (瘦AP)
*   **Fat AP (胖AP)**：
    *   **独立性**：全能型选手，自带OS，发射信号、管理用户、配置参数全靠自己。
    *   **缺点**：无法统一管理。如果有100个胖AP，必须逐台配置。
    *   **适用**：家庭、小型办公网络。
*   **Fit AP (瘦AP)**：
    *   **独立性**：提线木偶，不具备独立管理能力，必须由 **AC 控制器** 统一管控。
    *   **优点**：集中管理，支持漫游。只需在 AC 修改配置，所有 AP 自动同步。
    *   **适用**：大型企业、园区、校园网络。

### 1.2 AC 控制器 & CAPWAP 隧道
*   **AC 控制器**：无线网络的“中央大脑”。负责管理所有的瘦AP（SSID、功率、信道、用户接入）。
*   **CAPWAP 隧道**：AC 与 AP 之间的加密数据通道。
    *   **控制隧道**：用于下发配置、监控 AP 状态。
    *   **数据隧道**：用于回传用户上网数据（仅在隧道转发模式下使用）。

## 二、 WLAN 组网与转发模式 (核心领会)

### 2.1 组网模式
*   **二层组网**：AC 和 AP 在同一个 VLAN/广播域。AP 启动后通过二层广播直接发现 AC。
*   **三层组网 (重点)**：AC 和 AP 不在同一个网段。
    *   **Option 43**：核心考点。AP 在获取 IP 时，DHCP 服务器通过 Option 43 字段告知其 AC 的 IP 地址，解决跨网段发现问题。
*   **直连组网**：AC 在数据转发的主干道上。
*   **旁挂组网 (推荐)**：AC 挂在核心/汇聚交换机侧面。优点是网络更灵活，AC 故障不影响原有有线网络业务。

### 2.2 转发模式
*   **直接转发 (本地转发)**：用户数据流量在 AP 剥离标签后直接进有线网。AC 只管信令，转发效率高，AC 无压力。
*   **隧道转发 (集中转发)**：所有用户数据必须钻进 CAPWAP 隧道传回 AC，由 AC 统一转发。方便安全过滤，但 AC 压力巨大。

## 三、 WLAN 业务配置逻辑 (核心应用 - 必背)

华为配置遵循“模板化”逻辑，步骤如下：

1.  **基础准备**：创建 VLAN、配置接口 IP、配置 DHCP 地址池（给 AP 分配 IP，给终端分配业务 IP）。
2.  **设置 AC 源接口**：`capwap source interface vlanif [ID]`。
3.  **配置 AP 上线**：添加 AP 到管理范围（MAC 或 SN 认证），分配 AP 组（AP Group）。
4.  **配置三要素模板**：
    *   **Security Profile (安全模板)**：配置密码、加密算法。
    *   **SSID Profile (SSID 模板)**：起 WiFi 名字。
    *   **VAP Profile (VAP 模板)**：核心模板，绑定上述两个模板，并指定业务 VLAN。
5.  **应用配置**：进入 AP 组（或特定 AP）的射频视图（Radio），绑定 VAP 模板。

## 四、 关键配置命令 (考前速记)

### 1. AC 管理基础
```shell
# 指定AC源接口
[AC] capwap source interface vlanif 100  # 核心命令：必须与AP建立连接

# AP上线认证
[AC-wlan-view] ap-id 1 type-id 35 mac-address xxxx-xxxx-xxxx # 离线增加AP
[AC-wlan-view] ap-group name office # 创建AP组
```

### 2. 核心模板三部曲
```shell
# (1) 创建安全模板
[AC-wlan-view] security-profile name sec_p1
[AC-wlan-sec-prof-sec_p1] security wpa2 psk pass-phrase 12345678 aes

# (2) 创建SSID模板
[AC-wlan-view] ssid-profile name ssid_p1
[AC-wlan-ssid-prof-ssid_p1] ssid MyWiFi_Guest

# (3) 创建VAP模板（绑定三要素）
[AC-wlan-view] vap-profile name vap_p1
[AC-wlan-vap-prof-vap_p1] security-profile sec_p1
[AC-wlan-vap-prof-vap_p1] ssid-profile ssid_p1
[AC-wlan-vap-prof-vap_p1] service-vlan vlan-id 20  # 指定终端上网的VLAN
```

### 3. 应用配置到射频
```shell
[AC-wlan-view] ap-group name office
[AC-wlan-ap-group-office] vap-profile vap_p1 wlan 1 radio all # 全体射频(2.4G+5G)生效
```

### 4. 三层组网 DHCP Option 43 (选考点)
```shell
[Router-ip-pool-ap_pool] option 43 hex 030c30312e3136382e31302e31 # 格式为: 03+长度+AC十六进制IP
```

## 五、 考纲重点提炼：哪些必须背？

1.  **核心三要素模板**：SSID Profile、Security Profile、VAP Profile。
2.  **CAPWAP 通信机制**：掌握隧道建立的过程（获取IP->发现AC->加固隧道->上线）。
3.  **DHCP 配置**：不仅要配置 AP 的 IP 池，还要配置无线终端（STA）的业务 IP 池。
4.  **转发路径分析**：能根据要求画出或描述直接转发和隧道转发下，数据包是如何流动的。
5.  **故障排障**：学会使用 `display ap all` 检查 AP 是否为 `nor` (Normal) 状态；`display vap all` 检查业务是否正常下发。
