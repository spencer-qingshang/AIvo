### 三、 简答题
16. 
    (1) 用户视图：`<Huawei>`，基础查看权限。
    (2) 系统视图：`[Huawei]`，全局配置权限。
    (3) 接口视图：`[Huawei-GigabitEthernet0/0/0]`，特定接口配置。
    (4) 协议视图：`[Huawei-ospf-1]`，特定协议配置。
17. 
    (1) IP 地址必须配合子网掩码才能区分网络位和主机位，从而判断两个 IP 是否处于同一网段。
    (2) “24” 代表子网掩码中连续的 “1” 的个数为 24 个，即 255.255.255.0。

### 四、 综合应用题
18. 
    <Huawei> system-view
    [Huawei] sysname GZ_Core
    [GZ_Core] interface GigabitEthernet 0/0/0
    [GZ_Core-GigabitEthernet0/0/0] ip address 10.1.1.1 255.255.255.0
    [GZ_Core-GigabitEthernet0/0/0] display this  (或退出到外面执行 display ip interface brief)
    [GZ_Core-GigabitEthernet0/0/0] return
    <GZ_Core> save
