# 项目2：运用交换机构建小型园区网络 (考试复习精简版)

## 一、 核心知识点：必背“识记”与“领会”

### 1. VLAN 基础与帧格式 (高频考点)
*   **VLAN 的核心作用**：在物理网络上逻辑划分广播域，隔离广播风暴，提高安全性和管理效率。
*   **802.1Q 帧格式**：在以太网帧源 MAC 地址后插入 **4 字节的 Tag**。VLAN ID 范围为 **1-4094**。
*   **划分方式**：最常用的是**基于端口划分**。

### 2. 交换机端口类型处理规则 (必背逻辑)
| 端口类型 | 接收报文处理 | 发送报文处理 | 常用场景 |
| :--- | :--- | :--- | :--- |
| **Access** | 接收无标签帧：打上接口 PVID 标签。 | 剥离标签，发送原始帧。 | 连接 PC、服务器。 |
| **Trunk** | 若标签在允许列表中：直接接收；若无标签：打上 PVID。 | 若标签与 PVID 相同：剥离发送；不同且在允许列表：带标签发送。 | 连接交换机/路由器。 |
| **Hybrid** | 华为默认，可灵活控制。 | 决定是否剥离标签（Tagged/Untagged）。 | 连接终端或网络设备。 |

### 3. 三层交换与路由 (核心辨析)
*   **二层 vs 三层**：二层交换机看 MAC（数据链路层）；三层交换机看 IP（网络层）。
*   **VLANIF 接口**：三层交换机的虚拟接口，作为各 VLAN 的**网关**，实现不同 VLAN 间的路由转发。
*   **三层交换机 vs 路由器**：三层交换机硬件转发效率更高（一次路由，多次交换），但广域网接口支持不如路由器。

### 4. 链路聚合与生成树 (稳定性)
*   **Eth-Trunk (链路聚合)**：增加带宽、提高可靠性（链路冗余）。
*   **STP/RSTP (生成树)**：防止**网络环路**。原理是阻塞冗余端口。RSTP 是 STP 的快速演进版，收敛更快。

### 5. 高可用与自动配置 (VRRP/DHCP)
*   **VRRP (虚拟路由冗余协议)**：将多台网关设备虚拟成一台。关键术语：**Master**（主）、**Backup**（备）、**虚拟 IP**。优先级越大越优（默认 100）。
*   **DHCP (动态主机配置协议)**：自动分配 IP。池类型：**全局地址池**（灵活）和**接口地址池**（简单）。

## 二、 模拟试题预测 (考试怎么出)

### 1. 单项选择题 (考察概念)
*   **Q**: VLAN 技术主要解决了局域网中的哪个问题？
    *   A. 带宽不足  B. **隔离广播域**  C. IP 地址耗尽  D. 物理环路
*   **Q**: 交换机端口类型中，华为设备默认的是？
    *   A. Access  B. Trunk  C. **Hybrid**  D. Console

### 2. 填空题 (考察精准度)
*   Trunk 端口在发送报文时，如果 VLAN ID 与端口的 \_\_\_ 相同，则会剥离标签发送。 (答案：**PVID**)
*   实现 VLAN 间通信，在三层交换机上需要创建 \_\_\_ 虚拟接口。 (答案：**Vlanif**)
*   VRRP 默认优先级是 \_\_\_，数值越 \_\_\_ 越优先成为 Master。 (答案：**100**、**大**)

### 3. 简答题 (考察逻辑)
*   **Q**: 简述链路聚合 (Eth-Trunk) 的优势。
    *   **参考答案**：1. **增加带宽**：将多个物理链路带宽叠加；2. **提高可靠性**：其中一条物理链路故障，流量可自动切换到其他成员链路。

### 4. 综合应用题 (笔试答题模板)
*   **场景**：在 Core-SW 上实现 VLAN 10 与 VLAN 20 互通。
*   **答题思路**：
    ```shell
    # 1. 开启全局功能
    [Core-SW] vlan batch 10 20
    
    # 2. 配置 VLAN 10 网关
    [Core-SW] interface Vlanif 10
    [Core-SW-Vlanif10] ip address 192.168.10.254 24
    
    # 3. 配置 VLAN 20 网关
    [Core-SW] interface Vlanif 20
    [Core-SW-Vlanif20] ip address 192.168.20.254 24
    ```

## 三、 考前最后 5 分钟 (必杀技)

1.  **剥洋葱规则**：Access 出来必无标，Trunk 出来通常带标。
2.  **配置三部曲**：VLAN 配置一定要记：**建 VLAN -> 设类型 -> 划 VLAN**。
3.  **STP 记忆法**：数值越小越优先（Root 根网桥）。
4.  **VRRP 记忆法**：数值越大越优先（Master 主网关）。
5.  **排错技巧**：
    *   `display vlan`：查看 VLAN 划分和端口状态。
    *   `display stp brief`：查看哪个接口被阻塞了（Discarding）。
    *   `tracert`：追踪路径，判断是在哪跳断开的。
