# Track: 补充项目5大纲及生成WLAN实操手册 (spec.md)

## 1. 概述
本项目旨在完善 "项目5 构建无线的园区网络.md" 的大纲内容，将其转化为通俗易懂的“大白话”讲解。同时，根据大纲中的实验要求，生成两份详细的实操手册，分别对应“直连式二层WLAN”和“旁挂式三层WLAN”两种组网场景。

## 2. 功能要求

### 2.1 补充理论大纲 (项目5 构建无线的园区网络.md)
*   **输入**：现有的 `项目5 构建无线的园区网络.md` 文件。
*   **内容补充**：
    *   **直连式二层WLAN**：
        *   解释 Fat AP 与 Fit AP 的区别（大白话：一个是独立的智能手机，一个是需要中央大脑控制的提线木偶）。
        *   解释 AC 控制器的作用。
        *   解释 CAPWAP 隧道（AC 和 AP 之间的专线）。
    *   **旁挂式三层WLAN**：
        *   解释什么叫“旁挂”（AC 像个顾问挂在旁边，不直接抗流量）。
        *   解释什么叫“三层漫游”（走到哪 IP 都不变）。
    *   **关键技术应用**：
        *   解释 SSID（WIFI名字）、VAP（虚拟AP）等概念。
*   **风格**：通俗易懂，用生活中的例子类比（大白话）。

### 2.2 生成实操手册 1：直连式二层WLAN
*   **文件名**：`项目5_1_直连二层WLAN实战.md`
*   **拓扑**：AC6005 + AP + PC（模拟无线终端）。
*   **关键配置**：
    *   AC 作为 DHCP 服务器（为 AP 和 STA 分配 IP）。
    *   配置 AP 上线（认证方式：MAC认证或不认证）。
    *   配置 WLAN 业务（SSID: "Huawei-Layer2", 密码: "huawei@123"）。
    *   配置数据转发模式：直接转发。
*   **验证**：PC 能搜到 WIFI 并获取 IP。

### 2.3 生成实操手册 2：旁挂式三层WLAN
*   **文件名**：`项目5_2_旁挂三层WLAN实战.md`
*   **拓扑**：AC6005（旁挂） + CoreSW（核心交换机） + AccSW（接入交换机） + AP。
*   **关键配置**：
    *   CoreSW 作为 DHCP 服务器（管理 VLAN 和 业务 VLAN）。
    *   路由配置：AC 与 CoreSW 之间跑静态路由或 OSPF，确保 AC 能 ping 通 AP。
    *   配置 Option 43：告诉 AP，AC 的 IP 地址在哪里（这是三层组网的关键）。
    *   配置隧道转发模式（Tunnel Forwarding），流量统一回传给 AC 集中处理。
*   **验证**：AP 跨网段上线，STA 获取业务网段 IP 并能通网。
