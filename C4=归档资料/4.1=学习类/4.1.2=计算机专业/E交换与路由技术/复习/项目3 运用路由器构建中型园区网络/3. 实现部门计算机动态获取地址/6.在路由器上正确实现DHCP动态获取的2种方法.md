# 核心考点笔记：路由器 DHCP 两种实现方法的标准配置模板 (华为 VRP)

**⚠️ 绝对前置条件（必敲）：**

Plaintext

```
<Huawei> system-view
[Huawei] dhcp enable       // 在系统视图下，全局开启 DHCP 服务
```

---

## 方法一：基于接口的 DHCP 实现 (极简模式)

**适用场景：** 路由器（如 R 1）的 `GE0/0/0` 接口直连一个部门的交换机，直接把该接口所在的网段全部分配下去。

**配置逻辑：** 进接口 -> 配网关 IP -> 开启接口 DHCP -> (可选) 配置高级参数。

### 标准配置代码：

Plaintext

```
[Huawei] interface GigabitEthernet 0/0/0
[Huawei-GigabitEthernet0/0/0] ip address 192.168.1.254 24      // 配置接口IP，这会自动成为下方 PC 的默认网关
[Huawei-GigabitEthernet0/0/0] dhcp select interface            // 核心指令：告诉接口使用自己的网段分配 IP

// 以下为考试常考的附加配置（可选）：
[Huawei-GigabitEthernet0/0/0] dhcp server dns-list 114.114.114.114 8.8.8.8    // 下发 DNS 服务器地址
[Huawei-GigabitEthernet0/0/0] dhcp server excluded-ip-address 192.168.1.1 192.168.1.10 // 排除 1~10 这几个 IP 不参与分配（留给打印机等）
[Huawei-GigabitEthernet0/0/0] dhcp server lease day 3 hour 12                 // 修改租期为 3 天 12 小时（默认是 1 天）
[Huawei-GigabitEthernet0/0/0] quit
```

## 方法二：基于全局地址池的 DHCP 实现 (高级模式)

**适用场景：** 配合单臂路由、三层交换机，或者 DHCP 中继使用。需要高度自定义网段和网关。

**配置逻辑：** 建水池 -> 设网段/网关/DNS -> 进物理/逻辑接口 -> 挂载水池。

### 标准配置代码：

假设我们要建立一个名为 `VLAN10_Pool` 的地址池，网段为 `192.168.10.0/24`。

**第一步：配置全局地址池参数**

Plaintext

```
[Huawei] ip pool VLAN10_Pool                                // 创建全局地址池
[Huawei-ip-pool-VLAN10_Pool] network 192.168.10.0 mask 24   // 宣告要分配的网段和掩码
[Huawei-ip-pool-VLAN10_Pool] gateway-list 192.168.10.254    // 必须手动指定发给 PC 的默认网关
[Huawei-ip-pool-VLAN10_Pool] dns-list 8.8.8.8               // 指定 DNS
[Huawei-ip-pool-VLAN10_Pool] excluded-ip-address 192.168.10.250 192.168.10.253 // 排除特定 IP
[Huawei-ip-pool-VLAN10_Pool] quit
```

**第二步：在对应接口上调用该地址池**

Plaintext

```
[Huawei] interface GigabitEthernet 0/0/1                    // 进入需要下发该网段 IP 的接口 (也可能是子接口如 G0/0/1.10)
[Huawei-GigabitEthernet0/0/1] ip address 192.168.10.254 24  // 接口自身的 IP 必须和地址池的网关一致！
[Huawei-GigabitEthernet0/0/1] dhcp select global            // 核心指令：告诉该接口去全局池子里拿 IP 分配
[Huawei-GigabitEthernet0/0/1] quit
```

## 终极排错与验证步骤 (上机考试必用)

敲完代码后，如何确认自己 100% 拿到了这部分的分数？

**1. 在路由器上检查池子状态：**

- `display ip pool`
    
    _(查看所有地址池的简表，看有没有建成功)_
    
- `display ip pool interface GigabitEthernet 0/0/0` 或 `display ip pool name VLAN10_Pool`
    
    _(查看具体池子的详细信息，重点看 `Used` 字段，如果有数字，说明有电脑成功拿到 IP 了！)_
    

**2. 在终端 (PC) 上验证 (eNSP 操作)：**

- 双击 eNSP 里的 PC 终端，在“基础配置”页面，将 IPv 4 配置方式选为 **“DHCP”**。
    
- 切换到“命令行”页面，输入：
    
    `ipconfig` _(查看是否成功获取到了预期的 IP、掩码和网关)_
