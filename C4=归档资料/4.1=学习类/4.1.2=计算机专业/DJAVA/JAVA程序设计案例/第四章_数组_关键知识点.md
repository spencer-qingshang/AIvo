# 第四章：数组 - 必备关键知识点

本章涵盖Java中数组的基本概念、定义、操作和 `Arrays` 工具类的使用，是存储和处理同类型数据的基础。

---

## 1. 数组基本概念

*   **概念**：数组是一种**容器**，用于存储**同种数据类型**的多个值。
*   **核心特点 (必记)**：
    *   **定长**：数组一旦创建，其**长度不可改变**。
    *   **同质**：所有元素的类型必须相同。
    *   **引用类型**：数组本身是引用数据类型（即使存储的是基本数据类型）。
*   **索引/下标 (Index)**：
    *   数组中的每个元素都有一个编号，**从 `0` 开始**。
    *   最大下标是 `数组长度 - 1`。
    *   通过 `数组名[索引]` 访问元素。
*   **长度**：使用 `数组名.length` 属性获取（注意：**无括号**）。

---

## 2. 一维数组

### 2.1 定义与初始化

*   **定义格式**：`数据类型[] 数组名;` (推荐) 或 `数据类型 数组名[];`
*   **初始化方式**：
    *   **静态初始化** (已知具体值)：`int[] arr = {10, 20, 30};` (或 `new int[]{10, 20, 30};`)
    *   **动态初始化** (只知长度)：`int[] arr = new int[5];` (元素会有默认值：`int`为0，`double`为0.0，`boolean`为`false`，引用类型为`null`)

### 2.2 基本操作

*   **访问元素**：`arr[索引]` (读取或修改)。
*   **遍历**：
    ```java
    for (int i = 0; i < arr.length; i++) {
        System.out.println(arr[i]);
    }
    ```
    *   `foreach` 遍历 (增强for循环)：
    ```java
    for (int element : arr) {
        System.out.println(element);
    }
    ```

---

## 3. 二维数组

*   **概念**：**数组的数组**。二维数组的每个元素又是一个一维数组。
*   **形象理解**：类似表格或矩阵，由**行 (Row)** 和 **列 (Column)** 组成。
*   **访问方式**：需要两个下标 `数组名[行索引][列索引]`。
*   **长度概念 (易错点)**：
    *   `arr.length`：表示二维数组的**行数**。
    *   `arr[i].length`：表示第 `i` 行的**列数** (该行一维数组的长度)。

### 3.1 定义与初始化

*   **动态初始化**：`数据类型[][] 数组名 = new 数据类型[行数][列数];`
*   **静态初始化**：
    ```java
    int[][] arr = {
        {1, 2, 3},
        {4, 5, 6}
    };
    ```
### 3.2 遍历

*   使用**嵌套循环**遍历：外层循环遍历行，内层循环遍历列。
    ```java
    for (int i = 0; i < arr.length; i++) { // 遍历行
        for (int j = 0; j < arr[i].length; j++) { // 遍历当前行的列
            System.out.print(arr[i][j] + " ");
        }
        System.out.println(); // 每行结束换行
    }
    ```

---

## 4. `Arrays` 工具类 (`java.util.Arrays`)

*   **概念**：操作数组的**工具类**，提供了大量静态方法。
*   **常用方法**：
    *   `Arrays.toString(数组)`：将数组内容转换为字符串格式打印（例如 `[1, 2, 3]`）。
    *   `Arrays.sort(数组)`：对数组进行**升序**排序。
    *   `Arrays.binarySearch(数组, 查找值)`：二分查找（**前提：数组必须有序**）。
