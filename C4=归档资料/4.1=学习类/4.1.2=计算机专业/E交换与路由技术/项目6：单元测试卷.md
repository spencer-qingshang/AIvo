## 四、 简答与计算题 (共 15 分)

1.  **简述 ACL 通配符（Wildcard Mask）中“0”和“1”的逻辑含义。** (7分)
    *   答：0 表示“匹配”，即对应的二进制位必须严格一致；1 表示“忽略”，即对应的二进制位可以是 0 也可以是 1，不进行检查。

2.  **简述在园区网络出口配置 NAT（网络地址转换）的主要优点（至少写出三点）。** (8分)
    *   答：1. 节省公网 IP 地址资源；2. 隐藏内部网络拓扑，提高安全性；3. 解决内网地址重叠冲突问题。

## 五、 配置与综合案例分析题 (共 30 分)

1.  **ACL 配置演练：** (15分)
    ```shell
    [Router] acl 3005
    [Router-acl-adv-3005] rule 5 deny tcp source 192.168.1.0 0.0.0.255 destination 10.1.1.100 0 destination-port eq 80
    [Router-acl-adv-3005] quit
    [Router] interface GigabitEthernet 0/0/0
    [Router-GigabitEthernet0/0/0] traffic-filter inbound acl 3005
    ```

2.  **AAA 与 远程管理：** (15分)
    ```shell
    # AAA 配置
    [R1] aaa
    [R1-aaa] local-user admin password cipher Huawei@123
    [R1-aaa] local-user admin service-type telnet
    [R1-aaa] local-user admin privilege level 15
    [R1-aaa] quit
    
    # 应用到 VTY 线路
    [R1] user-interface vty 0 4
    [R1-ui-vty0-4] authentication-mode aaa
    ```
