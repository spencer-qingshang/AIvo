# 第二章：Java编程基础 - 必备关键知识点

本章是Java语言的入门，涵盖了程序中最基本、最核心的组成部分和操作规则。

---

## 1. Java关键字与标识符

*   **关键字 (Keywords)**：
    *   **概念**：Java语言中被赋予了**特殊含义**的单词。
    *   **特点**：全部是**小写**；**不能**用作标识符。
    *   **保留字**：`goto`, `const` (目前未使用，但不能用作标识符)。
*   **标识符 (Identifiers)**：
    *   **概念**：用于给变量、方法、类等命名的符号。
    *   **语法规范 (必记)**：
        *   只能由**字母、数字、下划线(`_`)、美元符号(`$`)**组成。
        *   **不能以数字开头**。
        *   **不能是关键字**。
        *   **严格区分大小写** (`myVar` 和 `MyVar` 是不同的)。
    *   **命名风格 (规范)**：
        *   **包名**：全小写。
        *   **类名/接口名**：**大驼峰** (每个单词首字母大写，如 `ArrayList`)。
        *   **变量名/方法名**：**小驼峰** (第一个单词首字母小写，后面单词首字母大写，如 `getMaxScore`)。
        *   **常量名**：全大写，单词间用**下划线**隔开 (如 `MAX_VALUE`)。

---

## 2. 常量与变量

*   **常量 (Constants)**：
    *   **概念**：程序运行过程中，**值固定不变**的量。
    *   **关键字**：`final`。
    *   **赋值**：只能被赋值一次。
    *   **字面值常量**：直接写出的值（如 `100`, `3.14`, `'A'`, `"Hello"`, `true`, `false`, `null`）。
*   **变量 (Variables)**：
    *   **概念**：程序运行过程中，**值可以发生改变**的量。
    *   **本质**：内存中的一小块区域。
    *   **三要素 (必记)**：**数据类型**、**变量名**、**值**。
    *   **声明与赋值**：`数据类型 变量名 = 值;` (局部变量使用前必须初始化)。

---

## 3. 基本数据类型

*   **分类**：Java数据类型分为**基本数据类型** (8种) 和 **引用数据类型**。
*   **8种基本数据类型 (必记)**：
    *   **整型**：`byte`(1字节, -128~127), `short`(2字节), `int`(4字节, 最常用), `long`(8字节)。
    *   **浮点型**：`float`(4字节, 单精度), `double`(8字节, 双精度, 默认)。
    *   **字符型**：`char`(2字节, 'A', 可存汉字)。
    *   **布尔型**：`boolean`(只有`true`/`false`)。
*   **后缀规则 (易错点)**：
    *   `long`类型数值末尾加 `L` 或 `l` (如 `100L`)。
    *   `float`类型数值末尾加 `F` 或 `f` (如 `3.14f`)。
*   **字符型 `char`**：底层是数字（Unicode），可参与运算。
*   **数据类型转换**：
    *   **自动类型转换 (隐式)**：**小范围 -> 大范围**。安全，系统自动完成。
        *   `byte/short/char` -> `int` -> `long` -> `float` -> `double`。
    *   **强制类型转换 (显式)**：**大范围 -> 小范围**。有风险（可能丢失精度或溢出），需手动 `(目标类型)值`。
    *   **表达式自动转换规则**：
        *   `byte/short/char` 参与运算时，一律先自动转换为 `int` 再计算。
        *   表达式的最终结果类型取决于参与运算的**最高级**类型。

## 4. 数据输入与输出

*   **`System.out` 类**：
    *   **作用**：标准输出流，向控制台打印数据。
    *   **方法**：`System.out.print()` (不换行), `System.out.println()` (换行)。
    *   **字符串拼接**：使用 `+` 运算符。
*   **`Scanner` 类 (`java.util.Scanner`)**：
    *   **作用**：文本扫描器，从控制台（键盘）获取用户输入。
    *   **使用“三部曲”**：
        1.  **导包**：`import java.util.Scanner;`
        2.  **创建对象**：`Scanner sc = new Scanner(System.in);`
        3.  **接收数据**：`sc.nextInt()`, `sc.nextDouble()`, `sc.next()`, `sc.nextLine()`。
    *   **`next()` vs `nextLine()`**：
        *   `next()`：遇到空格、Tab或回车停止读取。
        *   `nextLine()`：遇到回车停止读取，读取整行。
    *   **陷阱 (必记)**：`nextInt()` 后紧跟 `nextLine()` 时，`nextInt()` 读取数字后，留下的回车符会被 `nextLine()` 立即读取，导致跳过输入。**解决方案**：在`nextInt()`后加一个`sc.nextLine()`来“吃掉”回车符。

---

## 5. 运算符与表达式

*   **算术运算符**：`+`, `-`, `*`, `/`, `%` (取模), `++` (自增), `--` (自减)。
    *   **注意点**：整数除法结果是整数；取模结果符号与被除数相同；`++` 和 `--` 的前置（先变后用）与后置（先用后变）区别。
*   **赋值运算符**：`=`, `+=`, `-=`, `*=`, `/=`, `%=`。
    *   `a += b` 等价于 `a = a + b` (复合赋值运算符会自带隐式类型转换)。
*   **关系运算符**：`>`, `<`, `>=`, `<=`, `==` (相等判断), `!=` (不等判断)。
    *   结果为 `boolean` 类型。
*   **逻辑运算符**：`&&` (短路与), `||` (短路或), `!` (非), `&` (逻辑与), `|` (逻辑或)。
    *   **短路效应 (必记)**：`&&` 左边为 `false` 则右边不执行；`||` 左边为 `true` 则右边不执行。
*   **条件运算符 (三元运算符)**：`条件 ? 表达式1 : 表达式2`。
