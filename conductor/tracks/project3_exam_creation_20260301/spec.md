# Specification: 制作项目 3 单元测试卷 (Track: project3_exam_creation_20260301)

## Overview
参考 `项目2：单元测试卷.md` 的格式与题型，为“项目3：运用路由器构建中型园区网络”制作一份标准的单元测试卷。该试卷将涵盖路由器基础配置、DHCP、单臂路由、静态路由及动态路由协议（RIPv2/OSPF）等核心知识点。

## Functional Requirements
1. **内容覆盖：** 必须完整覆盖项目 3 大纲中的 8 个核心模块：
   - 模拟器设置与 Console 登录。
   - 路由器基本配置命令。
   - DHCP 两种配置方法（接口地址池/全局地址池）。
   - 单臂路由原理与配置。
   - 静态路由、默认路由及浮动路由。
   - RIPv2 动态路由配置与原理。
   - OSPF 单区域配置与原理。

2. **题型结构（参考项目 2）：**
   - **一、单项选择题：** 20 道（涵盖原理与基础命令）。
   - **二、填空题：** 10 道（侧重术语、参数、关键步骤）。
   - **三、简答题：** 5 道（考察工作原理、协议对比、故障分析）。
   - **四、综合应用题：** 2 道（侧重配置逻辑设计与 CLI 命令序列）。

3. **格式要求：**
   - 采用 Markdown 编写。
   - 题答分段：试卷主体仅包含题目，答案统一放在文末“参考答案”部分。
   - 代码块：配置命令需使用 `bash` 或 `shell` 代码块包裹。

## Non-Functional Requirements
- **准确性：** 所有考点必须符合华为 VRP 平台（eNSP 模拟器）的操作逻辑。
- **一致性：** 风格、用语需与项目 2 测试卷保持一致。
