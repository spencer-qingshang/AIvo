# 向往的四级词汇 - 第一单元 (Unit 1)

**核心记忆法**：词根词缀法 (Root/Affix)、谐音法 (Homophony)、联想法 (Association)、行近对照法 (Comparison)。

## 1. 旋转与宇宙 (Vers/Vert = Turn)
- **Universe** /juːnɪvɜːs/
    - *记忆*: **Uni** (联合/统一) + **Verse** (旋转) -> 联合起来旋转 -> **宇宙**。
- **Diverse** /daɪˈvɜːs/
    - *记忆*: **Di** (两个/不同) + **Verse** (旋转) -> 向两个方向转 -> **不同的，多种多样的**。
- **Convert** /kənˈvɜːt/
    - *记忆*: **Con** (全部/一起) + **Vert** (旋转) -> 彻底转过去 -> **转变，转化**。
- **Evolve** /ɪˈvɒlv/
    - *记忆*: **E** (Out/向外) + **Volve** (Roll/转) -> 沃尔沃(Volvo)车轮飞速旋转 -> 转出来 -> **进化，发展**。
- **Revolve** /rɪˈvɒlv/
    - *记忆*: **Re** (Repeat/重复) + **Volve** -> 来回转 -> **旋转**。
- **Involve** /ɪnˈvɒlv/
    - *记忆*: **In** (In/里面) + **Volve** -> 转进去了 -> **卷入，涉及**。

## 2. 土地与自然
- **Landscape** /ˈlændskeɪp/
    - *记忆*: **Land** (陆地) + **Scape** (看) -> 陆地上看猴子(Ape) -> 有猴子的地方都是风景 -> **风景，景色**。
- **Island** /ˈaɪlənd/
    - *记忆*: 谐音“爱尔兰” -> 岛国 -> **岛**。
- **Shore** /ʃɔː/
    - *记忆*: **Short** (短的) -> 伸入海里的短陆地 -> **海岸，海滨**。
- **Lawn** /lɔːn/
    - *记忆*: **Law** (Lawyer/律师) + **n** (门) -> 律师家门口种满草地 -> **草坪，草地**。
- **Slope** /sləʊp/
    - *记忆*: **Slow** (慢) + **pe** (爬) -> 慢慢爬坡 -> **斜坡**。
- **Skyline** /ˈskaɪlaɪn/
    - *记忆*: **Sky** (天空) + **Line** (线) -> 天空的轮廓线 -> **轮廓线**。
- **Rural** /ˈrʊərəl/
    - *记忆*: 谐音“弱弱” -> 农村留守的多是老弱 -> **乡村的**。
- **Breeze** /briːz/
    - *记忆*: 品牌“清风”纸巾 -> **微风**。

## 3. 物质与微粒
- **Substance** /ˈsʌbstəns/
    - *记忆*: **Sub** (下面) + **Stance** (Stand/站) -> 站在最下面的东西 -> **本质，实质** (引申为物质)。
- **Atom** /ˈætəm/
    - *记忆*: 谐音“阿童木”(铁臂小原子) -> **原子**。
- **Particle** /ˈpɑːtɪkl/
    - *记忆*: **Part** (部分) + **icle** (小) -> 很抠门抠下来的一小部分 -> **微粒，粒子**。
- **Fluid** /ˈfluːɪd/
    - *记忆*: **Flu** (Fly/飞流) + **id** -> 飞流直下一地 -> **液体，流动的**。
- **Dense** /dens/
    - *记忆*: 河南话“蹬死”(拉) -> 乱麻拉不开 -> **稠密的**。
- **Melt** /melt/
    - *记忆*: **Me** (我) + **lt** (Let/让) -> 我融化了她 -> **融化**。

## 4. 生命与细菌
- **Bacteria** /bækˈtɪəriə/
    - *记忆*: **Back** (背后) + **Teria** (特日/太阳) -> 背后特别没太阳 -> 容易滋生细菌 -> **细菌**。
- **Virus** /ˈvaɪərəs/
    - *记忆*: **Vi** (六) + **r** (花) + **us** (美国) -> 六朵花在美国(带不回来) -> 有病菌 -> **病毒，病菌**。
- **Germ** /dʒɜːm/
    - *记忆*: 谐音“折磨” -> 细菌折磨人 -> **细菌，微生物**。
- **Insect** /ˈɪnsekt/
    - *记忆*: **In** (里) + **Sect** (塞个他) -> 耳朵里塞个它 -> **昆虫**。

## 5. 动作与状态
- **Broad** /brɔːd/
    - *记忆*: **B** (不) + **Road** (路) -> 路不够宽 -> **宽阔的**。
- **Sheer** /ʃɪə/
    - *记忆*: **She** (她) + **er** (人) -> 她是个十足的人 -> **十足的，完全的**。
- **Locate** /ləʊˈkeɪt/
    - *记忆*: **Lo** (姥姥) + **Cate** (K他/打他) -> 姥姥要打他先要定位他在哪 -> **定位**。
- **Remote** /rɪˈməʊt/
    - *记忆*: **Re** (回) + **Mote** (Move/移动) -> 往回移动(离远点) -> **遥远的**。
- **Surround** /səˈraʊnd/
    - *记忆*: **Sur** (上面) + **Round** (圈) -> 在上面画圈 -> **包围，环绕**。
    - *Surrounding*: 环境。
- **Prevent** /prɪˈvent/
    - *记忆*: **Pre** (前) + **Vent** (Went/走) -> 走在你前面 -> **阻止**。
- **Sustain** /səˈsteɪn/
    - *记忆*: **Sus** (下) + **Tain** (拿/Hold) -> 从下面拿着/撑着 -> **维持，支撑**。
- **Maintain** /meɪnˈteɪn/
    - *记忆*: **Main** (主要) + **Tain** (拿) -> 主要都拿着 -> **维持，维修**。
- **Attain** /əˈteɪn/
    - *记忆*: **At** (加强) + **Tain** (拿) -> 一直想拿终于拿到 -> **获得**。
- **Entertain** /ˌentəˈteɪn/
    - *记忆*: **Enter** (进) + **Tain** (拿) -> 进来就拿着麦克风 -> **娱乐**。

## 6. 其他核心词
- **Cape** /keɪp/
    - *记忆*: 猎人用披风(Cape)抓住猿猴 -> **披风，海角**。
- **Gravity** /ˈɡrævəti/
    - *记忆*: **Grave** (坟墓/严肃) -> 官人爱我到坟墓 -> **重力，严重性**。
- **Spray** /spreɪ/
    - *记忆*: **S** (蛇) + **Pray** (趴在光线里) -> 蛇喷射毒液 -> **喷洒**。
- **Pray** /preɪ/
    - *记忆*: 趴在光线里 -> **祈祷**。
- **Spark** /spɑːk/
    - *记忆*: **S** (嘶嘶声) + **Park** (公园) -> 公园放烟花 -> **火花**。
- **Erode** /ɪˈrəʊd/
    - *记忆*: **E** (阿姨) + **Road** (路) -> 去阿姨家的路被坏了 -> **腐蚀，侵蚀**。
- **Source** /sɔːs/
    - *记忆*: 谐音“骚死” -> 找味道的源头 -> **源头**。
- **Mental** /ˈmentl/
    - *记忆*: **Men** (人) + **Tal** (头) -> 人头脑里的 -> **精神的**。
- **Bloom** /bluːm/
    - *记忆*: **6100** (bl) + **m** (花) -> 6100朵花 -> **盛开，繁荣**。
- **Bundle** /ˈbʌndl/
    - *记忆*: 谐音“巴豆” -> 一包巴豆 -> **捆，包**。
- **Derive** /dɪˈraɪv/
    - *记忆*: **De** (弟弟) + **River** (河) -> 弟弟沿河找源头 -> **起源于** (derive from)。
- **Deprive** /dɪˈpraɪv/
    - *记忆*: **De** (弟弟) + **Prive** (扑) -> 弟弟扑进河里捞金项链 -> **剥夺**。
- **Formation** /fɔːˈmeɪʃn/
    - *记忆*: **Form** (形成) -> **形成，构造**。
- **Habitat** /ˈhæbɪtæt/
    - *记忆*: **Habit** (习惯) + **at** (在这) -> 习惯在这住 -> **栖息地**。
- **Extinct** /ɪkˈstɪŋkt/
    - *记忆*: **Extinguisher** (灭火器) -> 火灭了 -> **灭绝的**。
- **Preserve** /prɪˈzɜːv/
    - *记忆*: **Pre** (提前) + **Serve** (服务) -> 提前留好食物 -> **保存，保护**。
- **Conserve** /kənˈzɜːv/
    - *记忆*: **Con** (大家) + **Serve** -> 为大家保存 -> **保存**。
- **Deserve** /dɪˈzɜːv/
    - *记忆*: **De** (弟弟) + **Serve** -> 弟弟为你服务是应该的 -> **应得**。
- **Reserve** /rɪˈzɜːv/
    - *记忆*: **Re** (再) + **Serve** -> 再次服务/预定房间 -> **预定，保留**。
- **Humid** /ˈhjuːmɪd/
    - *记忆*: **Hu** (胡了) + **Mid** (中间) -> 打麻将中间胡了激动的出汗 -> **潮湿的**。
- **Moist** /mɔɪst/
    - *记忆*: **Mo** (毛) + **ist** -> 长毛了 -> **潮湿的**。
- **Domestic** /dəˈmestɪk/
    - *记忆*: **Dome** (Home) -> 家 -> **家庭的，国内的**。
- **Intermediate** /ˌɪntəˈmiːdiət/
    - *记忆*: **Inter** (中间) + **Media** (媒体) -> **中间的**。
- **Fore** /fɔː/
    - *记忆*: **Four** (四) -> 四个阿姨冲在前面 -> **前面的** (Forehead 前额)。
    - *拓展*: Outward (向外), Backward (向后), Upward (向上), Downward (向下).
- **Clockwise** /ˈklɒkwaɪz/
    - *记忆*: **Clock** (钟) + **Wise** (明智) -> 像钟一样明智地走 -> **顺时针的**。
- **Apart** /əˈpɑːt/
    - *记忆*: **A** + **Part** (部分) -> 分成一个个部分 -> **分开的**。
- **Apartment** /əˈpɑːtmənt/
    - *记忆*: 一个个分开的门 -> **公寓**。
- **Department** /dɪˈpɑːtmənt/
    - *记忆*: **De** (分) + **Part** (部分) -> 分开的部门 -> **部门**。
- **Enclose** /ɪnˈkləʊz/
    - *记忆*: **En** (使) + **Close** (关) -> 使关起来 -> **围住**。
- **Sector** /ˈsektə/
    - *记忆*: **Se** (Separate/分) -> 分开的区域 -> **部门，行业**。
- **Foundation** /faʊnˈdeɪʃn/
    - *记忆*: **Found** (建立) -> **基础，地基，基金会**。
- **Strategy** /ˈstrætədʒi/
    - *记忆*: **St** (街) + **Rat** (老鼠) -> 对付街上的老鼠需要策略 -> **策略，战略**。
- **Tangible** /ˈtændʒəbl/
    - *记忆*: **Tan** (弹) -> 能弹到的/摸到的 -> **可触摸的，有形的**。
