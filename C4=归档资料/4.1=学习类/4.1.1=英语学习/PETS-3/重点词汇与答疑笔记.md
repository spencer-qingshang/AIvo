## 🏗️ PETS-3 真题 (2015年9月 - Reading Text 1)

### Q26: 背景信息题
- **解题点：** 第一段开头 `born... to parents who were slaves`。
- **核心词：**
    - `Slave`：奴隶。
    - `Slaveholder`：奴隶主。

### Q27: 原因分析题 (易错点)
- **解题点：** 第二段 `took the blow instead` (挨打) 和 `furious` (愤怒)。
- **核心词：**
    - `Blow`：殴打、打击。
    - `Furious`：大发雷霆的。

### Q28: 关键信息定位
- **关键词：** `Constitution` (宪法), `Equal` (平等)。
- **词汇：**
    - `Obey`：服从、听从。
    - `Servant`：仆人。

### Q29: 跨段落逻辑题
- **解题点：** 第四段 `went to work for Sedgewick`。
- **背景：** 需要结合第二段信息，确认 `Sedgewick` 是 `Lawyer` (律师)。

### 📅 2026-01-27 (新课: 48-146 句)
*记录今天高强度语法闯关的核心收获。*

| 单词/短语 | 含义 | 语法/语境 |
| :--- | :--- | :--- |
| **fawn over** | 奉承、讨好 | 众人围着婴儿夸奖的动作 |
| **it's all...** | 氛围突变 | 描述一种集体瞬间改变的行为 |
| **cream puff** | 泡芙 / 娘娘腔 | 米奇的自嘲与卡姆对零食的渴望 (双关) |
| **if it wasn't for...** | 要不是因为... | 虚拟语气，对现状的假设 |
| **knows no...** | 不分.../超越了... | Love knows no race... (排比句型) |
| **creed** | 信条、教义 | 宗教或政治信仰 |
| **mean to do sth.** | 打算/故意 | I didn't mean to (不是故意的) |
| **Excalibur** | 石中剑 | 文化隐喻：把救 Luke 比作拔圣剑 |
| **have sb. over** | 邀请某人来家里 | 动态跨越空间的感觉 (over) |
| **take it outside** | 去外面解决 | 换个地方打架/争论 |
| **might as well** | 还不如... | 无奈之下的次优选择 |
| **be bound to do** | 注定/一定会 | 形容词固定搭配 |
| **blowup** | 激烈争吵 | 名词，指情绪爆发 |
| **I'm just saying** | 我只是随口说说 | 缓冲语气，委婉建议 |
| **gonna / wanna** | 打算 / 想要 | 极高频口语缩写 |
| **follow through** | 贯彻到底 | 坚持执行约定 |
| **works for me** | 我觉得行 | 同意方案 |
| **dinner thing** | 那个晚餐什么的 | thing 表示模糊指代 |
| **show a house** | 带人看房 | 房产中介术语 |
| **dude** | 哥们/老兄 | 父子间随意的称呼 |
| **I live up here** | 我情绪很高昂 | 手势指着上面，比喻情绪水平 |

#### 🧠 核心语法逻辑回顾
- **时态呼应：** `I thought I could...` (主从句时态必须步调一致)。
- **不定式否定：** 永远是 `not to do`。
- **进行时表禁止：** `You are not giving the speech` (比 don't 语气更强硬)。
- **宾补结构：** `imagine one of them pregnant` (形容词作补足语)。
- **虚拟语气：** `You would have stopped... if you weren't staring...` (对过去的假设)。
