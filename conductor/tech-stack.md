# AIvo 技术栈

## 1. 核心媒介
- **Markdown:** 项目的所有知识记录、计划和复盘均基于 Markdown 语法。

## 2. 软件与平台
- **Obsidian:** 核心知识管理平台，利用其本地库、双向链接及插件生态进行管理。
- **Git:** 负责整个知识库的版本控制，确保学习记录的可追溯性。

## 3. 编程与脚本
- **Python:** 用于自动化处理笔记内容的脚本工具（如章节提取）。
- **Java:** 作为自考计算机专业的核心学习对象，库中包含相关的代码案例和技术笔记。

## 4. 关键插件生态
- **MarkMind:** 用于构建学科知识体系的脑图。
- **Obsidian-Git:** 实现笔记与远程仓库的自动/半自动同步。
- **Easy Typing:** 优化笔记录入体验。
