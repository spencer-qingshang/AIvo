# 2026年3月15日练习错题整理

## 1. 2019年4月真题 第4题：人物性格
**题目：** 下列描述中，表现了《秦晋殽之战》先轸这一人物性格特点的是：
**正确答案：** B. 多谋善断，忠直刚烈
**解析：** 先轸准确预判形势体现了多谋善断；在放走秦国三将时“不顾而唾”体现了忠直刚烈。

## 2. 2019年4月真题 第12题：意象比拟
**题目：** 下列《我与地坛》的句子，作者用以比作秋的是：
**正确答案：** B. 铜钟与这园子一般年纪，浑身挂满绿锈
**解析：** 铜钟浑身绿锈，展现了岁月沉淀与萧瑟深沉的秋意。

## 3. 2019年4月真题 第14题：抒情方法
**题目：** 《湘夫人》中“目眇眇兮愁予”“思公子兮未敢言”的抒情方法是：
**正确答案：** A. 直抒胸臆
**解析：** “愁予”、“思公子”直接表达内心忧愁和思念，未借助其他景物。

## 4. 2019年4月真题 第15题：艺术表现手法
**题目：** 《长恨歌》因情造景的诗句是：
**正确答案：** D. 楼阁玲珑五云起，其中绰约多仙子
**解析：** 因为深沉的思念（情），幻想出不存在的蓬莱仙岛（景），属于因情造景。

## 5. 2019年4月真题 第16题：修辞手法
**题目：** 下列《八声甘州》的词句中，运用拟人修辞手法的是：
**正确答案：** C. 惟有长江水，无语东流
**解析：** 将长江水比作人，“无语”赋予水人的状态，衬托孤寂。

## 6. 2019年4月真题 第18题：文学常识逻辑
**题目：** 《祖国啊，我亲爱的祖国》情感逻辑的第三个层次是：
**正确答案：** B. 为祖国的新生和美好未来而振臂欢呼
**解析：** 诗歌四层逻辑：历史负重、希望失落、新生欢呼、奉献献身。

## 7. 2019年4月真题 第20题：作品主题
**题目：** 《苦恼》中姚纳最想倾诉的苦恼是：
**正确答案：** C. 儿子不幸去世
**解析：** 全篇围绕姚纳想要找人倾诉丧子之痛而不得，凸显底层人的孤寂苦恼。
